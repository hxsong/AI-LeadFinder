## 模块说明

本模块涵盖从探针部署到动态线索完整入库的全链路功能，包含两个子阶段：

1. **P28-P38**：探针部署 → 访客行为采集 → 行为数据同步 → 动态评分 → 访客雷达列表
2. **P39-P46**：访客详情查看 → IP 解析识别 → 联系人挖掘 → 动态线索打分 → 动态线索池

## 关键业务规则

- **访客数据保留**：90 天归档，已挖掘访客永久保留（详见 `../07-全局规范/04-数据保留策略.md`）
- **Credits 消耗**：IP 解析固定 5 Credits/次，联系人按成功数扣费（详见 `../07-全局规范/02-Credits余额规则.md`）
- **AI 评分**：仅使用规则评分（不调用 LLM），降级链路见 `../07-全局规范/03-AI异常处理策略.md`
