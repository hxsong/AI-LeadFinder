## 功能目标

本模块涵盖从搜客需求输入到静态线索完整入库的全链路功能。核心目标包括：

1. **搜客需求输入**：用户通过自然语言补充短期搜客需求
2. **搜客策略生成**：AI 根据用户需求和 ICP 画像生成完整的调用策略
3. **搜客执行监控**：实时展示企业信息提取、ICP 匹配打分、企业信息增强全流程
4. **搜客记录管理**：展示历史搜客任务记录，支持状态筛选和重试
5. **搜客结果展示**：展示经过 AI 分析和 ICP 匹配打分排序的企业线索列表
6. **静态线索池**：管理静态数据获客来源的企业线索，承载联系人挖掘任务
7. **线索详情展示**：展示完整的线索分析报告，包含企业信息、评分、联系人矩阵

## 用户故事

### 故事一：企业管理员输入搜客需求

**作为** 企业管理员  
**我想要** 输入自然语言搜客需求并设置目标企业数量  
**以便于** 让 AI 根据我的需求找到匹配的潜在客户

**验收条件**：
- 可以输入最多 500 字符的自然语言需求
- 可以选择快速筛选标签
- 可以设置目标企业数量（20-200）
- 未输入需求时可以使用 ICP 画像默认搜索

### 故事二：企业管理员预览并执行搜客策略

**作为** 企业管理员  
**我想要** 预览 AI 生成的搜客策略和费用预估  
**以便于** 确认策略合理性后再执行

**验收条件**：
- 可以查看策略解读
- 可以查看瀑布流执行计划
- 可以查看费用预估明细
- 可以确认执行或返回修改

### 故事三：企业管理员监控搜客执行进度

**作为** 企业管理员  
**我想要** 实时查看搜客任务的执行进度  
**以便于** 了解当前处理状态和预计完成时间

**验收条件**：
- 可以查看企业信息提取进度
- 可以查看 ICP 匹配打分进度
- 可以查看企业信息增强进度
- 完成后自动跳转结果页面

### 故事四：企业管理员查看搜客记录

**作为** 企业管理员  
**我想要** 查看历史搜客任务记录  
**以便于** 追踪过往搜客情况和结果

**验收条件**：
- 可以按状态筛选记录
- 可以按时间筛选记录
- 可以查看每条记录的摘要和消耗
- 失败任务可以重试

### 故事五：企业管理员浏览搜客结果

**作为** 企业管理员  
**我想要** 浏览搜客结果并查看企业详情  
**以便于** 找到有价值的潜在客户

**验收条件**：
- 可以查看候选企业列表
- 可以按评分排序
- 可以筛选和导出结果
- 可以对感兴趣的企业进行深度挖掘

### 故事六：企业管理员挖掘联系人

**作为** 企业管理员  
**我想要** 对感兴趣的企业进行联系人挖掘  
**以便于** 获取企业的关键联系人信息

**验收条件**：
- 可以选择企业进行深度挖掘
- 可以查看挖掘进度
- 可以查看挖掘出的联系人信息
- 可以查看 Credits 消耗明细

## 业务规则

### 规则一：搜客需求输入规则

| 规则项 | 说明 |
|--------|------|
| 文本长度 | 最大 500 字符，超出部分自动截断 |
| 快速筛选 | 支持标签：美国/欧洲/有海关记录/近期参展/大型企业 |
| 目标数量 | 范围 20-200，默认 100 |
| 默认搜索 | 用户未输入时，自动使用 ICP 画像默认搜索 |

### 规则二：并发限制

**搜客任务并发限制**：

| 规则项 | 说明 |
|--------|------|
| 单任务限制 | 同一 BOT 最多 1 个搜客任务执行中 |
| 自动跳转 | 当前 BOT 已有搜客任务执行中时，进入页面自动跳转至搜客执行中页面 |

**联系人挖掘并发限制**：

| 规则项 | 说明 |
|--------|------|
| 并发限制 | 单租户最多 3 个联系人挖掘任务同时进行 |
| 超限处理 | 超出限制时需等待已有任务完成后方可执行 |

### 规则三：Credits 余额规则

| 规则项 | 说明 |
|--------|------|
| 余额范围 | 余额可为负，≤0 时禁止消耗操作 |
| 扣费时机 | 执行中不扣费，完成后按实际消耗结算 |
| 按钮控制 | 余额 ≤ 0 或余额 < 预估消耗时，确认搜客按钮禁用 |

### 规则四：策略生成规则

| 规则项 | 说明 |
|--------|------|
| 超时降级 | AI 策略生成超时（60s）降级返回预置策略模板 |
| 步骤限制 | 单次执行最多 4 个服务调用步骤 |
| 数量上限 | 目标企业数量上限 200 家，超出截断 |

### 规则五：四维评分计算

**计算公式**：

```
潜在客户得分 = 需求匹配度 × W1 
             + 购买能力 × W2 
             + 采购意向 × W3 
             + 活跃意向 × W4
```

**默认权重**：
- 需求匹配度(W1): 30%
- 购买能力(W2): 25%
- 采购意向(W3): 25%
- 活跃意向(W4): 20%

权重可根据业务类型动态调整，但总和必须等于 1.0。

**维度说明**：

| 维度 | 说明 |
|------|------|
| 需求匹配度 | AI 模型判断候选企业是否是客户的潜在客户 |
| 购买能力 | 企业规模评分（60%）+ 地理评分（40%） |
| 采购意向 | 海关 HS Code 匹配度 × 时间衰减系数 |
| 活跃意向 | 展会记录 × 时间衰减系数 |

### 规则六：评分等级划分

| 评分范围 | 等级 | 颜色 |
|---------|------|------|
| ≥80 | S级 | 绿色 |
| 60-79 | A/B级 | 蓝色 |
| <60 | C级 | 灰色 |

### 规则七：联系人挖掘计费

## 用户操作流程

### 场景一：输入搜客需求（P12）

1. 用户输入自然语言需求（最大 500 字符）
2. 可选点击快速筛选标签
3. 拖动滑块设置目标企业数量（默认 100，范围 20-200）
4. 点击"确认搜客"按钮提交
5. 系统生成任务并跳转至搜客策略预览页面

### 场景二：使用 ICP 画像默认搜索（P12）

1. 用户未输入任何文字，直接提交空表单
2. 系统自动使用 ICP 画像作为搜索指令
3. 系统生成任务并跳转至搜客策略预览页面

### 场景三：AI 策略生成与预览（P13-P16）

1. 系统根据用户需求和 ICP 画像生成策略
2. 页面展示三大模块：
   - 策略解读
   - 瀑布流执行计划
   - 费用预估明细
3. 用户确认策略后点击执行
4. 跳转至搜客执行中页面

### 场景四：搜客任务执行监控（P16-P18）

1. **企业信息提取阶段**：
   - 系统提取候选企业列表
   - 系统清洗数据：去重、补全字段、标准化格式
   - 展示候选企业基础信息

2. **ICP 匹配打分阶段**：
   - 系统对候选企业进行 AI 评分
   - 计算四维评分，加权得出最终得分
   - 过滤低于阈值的企业（默认 60 分）

3. **企业信息增强阶段**：
   - 系统对高评分企业补充数据
   - 补充海关数据、展会数据、地图数据

### 场景五：查看搜客记录（P17-P19）

1. 用户进入搜客记录页面
2. 系统展示搜客记录列表（默认按创建时间倒序）
3. 每条记录展示：搜索需求摘要、状态标签、创建时间、结果数量、消耗 Credits
4. 用户可以按状态筛选（全部/进行中/已完成/失败）
5. 用户可以按时间筛选（今日/近7天/近30天/全部）
6. 失败任务可执行重试（最多 3 次）

### 场景六：浏览搜客结果列表（P20-P22）

1. 用户进入搜客结果列表页面
2. 系统以卡片形式展示候选企业列表，默认按评分倒序
3. 每张卡片展示：匹配度环形评分图、企业名称、官网链接、徽章标签、企业简介、AI 推荐理由、员工规模
4. 用户可以筛选和排序
5. 用户可以导出结果为 Excel

### 场景七：联系人挖掘（P22-P27）

1. 用户在搜客结果列表点击"深度挖掘"按钮
2. 系统弹出联系人挖掘弹窗，展示计费规则
3. 用户确认后，系统开始执行挖掘任务
4. 用户跳转至静态线索池页面查看进度
5. 挖掘完成后，用户可点击线索卡片进入详情页查看完整联系人信息

### 场景八：查看静态线索池（P21-P27）

1. 用户进入静态线索池页面
2. 系统展示线索列表（默认按入库时间倒序）
3. 展示两类卡片：
   - 挖掘中状态：卡片边框高亮 + "挖掘中"标签 + 进度条
   - 已完成状态：展示联系人头像和姓名 + Credits 消耗
4. 用户可以筛选（挖掘状态、入库时间范围）
5. 用户可以排序（入库时间倒序/评分倒序）

### 场景九：查看线索详情（P26-P27）

1. 用户点击线索卡片进入详情页面
2. 系统展示：
   - 企业核心信息区（名称、域名、所在地、规模）
   - 最终线索评分与 AI 判词
   - 原始静态证据（海关数据、展会数据、地图数据、网页数据）
   - 联系人矩阵（姓名、职位、邮箱、验证状态、社媒链接）
   - 账单摘要（Credits 消耗明细）

## 异常处理

| 异常场景 | 系统行为 |
|---------|---------|
| Credits 余额不足 | 按钮禁用，提示充值 |
| ICP 画像不存在 | 按钮禁用，提示重新生成业务分析报告 |
| 已有搜客任务执行中 | 自动跳转至搜客执行中页面 |
| AI 策略生成超时 | 降级返回预置策略模板 |
| 任务不存在 | 跳转至初始化页面 |
| 结果数据为空 | 显示空态引导 |
| 挖掘已达上限 | 按钮禁用 |
| 线索已存在 | 提示不可重复入库 |
