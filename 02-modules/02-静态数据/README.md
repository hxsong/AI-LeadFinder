## 模块说明

本模块涵盖从搜客需求输入到静态线索完整入库的全链路功能，包含两个子阶段：

1. **P12-P21**：搜客需求输入 → AI 搜客策略生成 → API 调度抓取 → 四维评分 → 企业列表展示
2. **P22-P27**：从目标企业列表选择企业 → 联系人挖掘 → 线索乘数打分 → 静态线索池
