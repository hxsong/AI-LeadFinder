# AI-LeadFinder 产品需求文档（PRD）

> **版本**：v1.0.0
> **最后更新**：2026-04-10

---

## 文档目的

本文档是 AI-LeadFinder 产品的完整需求索引，涵盖从账号开通（P01）到全景触达（P80）的全链路产品需求。PRD 节点编号（P01-P80）与原型页面一一对应，供产品评审、研发实现和测试验收使用。

---

## 产品阶段总览

| 阶段 | PRD 节点 | 核心功能 |
|------|----------|---------|
| 01-账号与资源管理 | P01-P11 | 租户开通、Credits 资源、BOT 工作台、业务材料上传/独立站授权、AI 业务分析 |
| 02-静态数据获客 | P12-P27 | 搜客需求输入、AI 搜客策略、API 调度抓取、四维评分、静态线索池、联系人挖掘 |
| 03-动态流量获客 | P28-P46 | 探针部署部署、访客雷达、行为打分、动态线索池 |
| 04-全景画像与触达 | P47-P70 | 线索合并、触达渠道配置（邮件/LinkedIn/WhatsApp）、触达自动化（待补充原型） |
| 05-运营与合规 | P71-P80 | CRM 同步、标签体系、GDPR 合规、操作日志（待补充原型） |
| 06-非功能需求 | - | 性能指标、并发限制、安全要求（详见 `06-非功能需求/`） |
| 07-全局规范 | - | 全局响应码、Credits 规则、AI 异常处理、数据保留策略 |
