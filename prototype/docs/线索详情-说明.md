---

## 概述

线索详情页是静态线索挖掘流程的终局呈现页面，展示完整的线索分析报告。页面分为上下两层结构：上层展示企业核心信息、最终线索评分、AI 判词和联系人矩阵；下层展示该线索被收录的原始静态证据（海关数据、展会数据、地图数据、网页数据）。页面底部提供"使用智能触达 (ITA) 生成全渠道开发信"按钮，引导用户进入触达流程。

---

## 页面入口与出口

### 入口

| 来源 | 触发条件 | 携带参数 |
|------|---------|---------|
| 静态线索池.html | 用户点击线索卡片 | `bot_id`, `lead_id` |

### 出口

| 目标 | 触发条件 | 携带数据 |
|------|---------|---------|
| 静态线索池.html | 用户点击"返回线索列表"按钮 | `bot_id` |
| 智能触达 (ITA) | 用户点击"使用智能触达"主按钮 | `lead_id` |
| 智能触达 (ITA) | 用户点击联系人"发送邮件" | `lead_id`, `contact_email` |
| 外部链接 | 用户点击 LinkedIn/Twitter 链接 | URL（外部打开） |

## 功能逻辑

### 场景一：正常查看线索详情

**触发条件**：用户从静态线索池点击某条线索卡片进入。

**执行逻辑**：
1. 前端请求 `GET /api/v1/leads/{lead_id}` 获取完整报告
2. 页面展示企业核心信息区（名称、域名、所在地、规模）
3. 展示最终线索评分与 AI 判词
4. 展示联系人矩阵（姓名、职位、邮箱、验证状态、社媒链接）
5. 展示原始静态证据（海关数据、展会数据、地图数据、网页数据）
6. 展示账单摘要（Apollo.io Credits 消耗明细）

**输出**：完整线索报告页面。

---

### 场景二：AI 判词异常

**触发条件**：AI 模型返回空内容、格式错误、内容幻觉或字段缺失。

**执行逻辑**：
1. 前端检测 AI 判词字段异常状态
2. 按异常类型显示对应降级文案（详见异常处理章节）
3. 页面其余内容正常展示，不因 AI 判词异常而阻断

**输出**：线索报告页面，AI 判词区域显示降级文案。

---

### 场景三：无联系人结果

**触发条件**：联系人挖掘未返回有效联系人。

**执行逻辑**：
1. 联系人矩阵区域显示"未找到关键联系人"
2. 账单摘要显示消耗 0 Credits

**输出**：线索报告页面，联系人区域为空态展示。

---

### 场景四：数据缺失

**触发条件**：某类原始证据数据不存在。

**执行逻辑**：
1. 对应证据卡片显示"暂无数据"状态
2. 其余有数据的卡片正常展示

**输出**：线索报告页面，部分证据区显示空态。

## 页面状态

| 状态 | 触发条件 | UI 表现 |
|-----|---------|---------|
| 正常 | 数据完整返回 | 展示完整报告 |

## 边界规则

| 规则 | 条件 | 系统行为 |
|------|------|---------|
| 权限控制 | 当前版本无权限隔离 | 所有登录用户均可查看线索详情 |
| 数据只读 | 页面为只读模式 | 不支持编辑联系人信息 |
| 线索状态 | 无论挖掘是否成功 | 点击深度挖掘的线索均进入线索池，线索池中不显示评分字段 |

## 异常处理

| 异常场景 | 检测方式 | 前端表现 | Toast 文案 |
|---------|---------|---------|-----------|
| 线索不存在 | code 404 | 显示"该线索不存在或已被删除"提示，页面跳转回线索池 | 「该线索不存在或已被删除」 |
| 网络超时 | 请求超时 30s | 显示"线索详情加载失败" | 「网络连接异常，请检查网络后重试」 |
| 服务不可用 | code 500 | 显示"服务暂时不可用" | 「服务暂时不可用，请稍后再试」 |
| AI 判词为空 | ai_reasoning 为空 | 显示默认研判模板 | 「AI 研判结果为空」 |
| AI 格式错误 | JSON 解析失败 | 显示原始数据摘要 | 「AI 返回数据格式异常」 |
| AI 内容幻觉 | 与原始数据矛盾 | 标记"仅供参考" | 「AI 研判仅供参考，以实际数据为准」 |
| AI 字段缺失 | 必要字段为空 | 使用默认值填充 | 「AI 研判不完整」 |
| 邮箱无效 | 点击无效邮箱 | 提示邮箱地址无效 | 「该联系人的邮箱地址无效」 |
| 邮箱未验证 | 发送未验证邮箱 | 提示发送可能失败 | 「该联系人邮箱未经验证，发送可能失败」 |
| 浏览器限制 | 复制操作不支持 | 提示手动选择复制 | 「浏览器不支持自动复制，请手动选择复制」 |

## 字段映射

| 前端展示字段 | 后端字段名 | 数据来源说明 |
|-------------|-----------|------------|
| 线索 ID | lead_id | 系统生成，UUID |
| 企业名称 | enterprise_info.name | Apollo.io 或三方 API 返回 |
| 企业域名 | enterprise_info.domain | Apollo.io 返回 |
| 所在地 | enterprise_info.location | Apollo.io 返回 |
| 企业规模 | enterprise_info.employee_count | Apollo.io 返回 |
| 最终线索评分 | enterprise_info.final_lead_score | 打分引擎计算结果 |
| 线索等级 | enterprise_info.lead_grade | S/A/B/C，由评分映射 |
| AI 判词 | enterprise_info.ai_reasoning | LLM 生成，需做空值和格式校验 |
| 海关数据 | original_static_evidence.customs_data | 出海派 API 返回 |
| 展会数据 | original_static_evidence.exhibition_data | 10times API 返回 |
| 地图数据 | original_static_evidence.maps_data | Google Maps API 返回 |
| 网页数据 | original_static_evidence.website_data | Jina Reader API 返回 |
| 账单明细 | billing_receipt | Apollo.io Credits 消耗记录 |
| 联系人姓名 | contacts_matrix[].name | Apollo.io 返回 |
| 联系人邮箱 | contacts_matrix[].email | Apollo.io reveal 接口揭示 |
| 邮箱验证状态 | contacts_matrix[].email_verified | Apollo.io 返回 |
| 社媒链接 | contacts_matrix[].social_links | Apollo.io 返回 |
| 入库时间 | created_at | ISO 8601 时间戳 |
