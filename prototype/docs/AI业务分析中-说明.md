## 概述

AI 业务分析中页面是 AI 业务分析的核心处理页面，负责将原始输入数据（文档或独立站数据）转换为结构化的客户信息（ICP 画像）。系统通过异步任务队列处理大模型长耗时任务，前端实时展示两个阶段处理进度：P07 文本提取与结构化（ETL）、P08 深度分析与 ICP 建模。

## 功能逻辑

### 场景一：文档上传模式 - 文本提取与结构化（P07）

**触发条件**：用户从上传业务材料.html 进入。

**执行逻辑**：
1. 解析上传文件：PDF（LlamaParse/Unstructured.io）、WORD（python-docx）、MD/TXT（直接读取）
2. 输出带排版噪音的 Markdown 长文本
3. 调用 ETL 数据清洗 Prompt（gpt-4o-mini / Gemini 1.5 Flash，强制开启 JSON Mode）
4. AI 严格按 JSON Schema 输出干净的实体字典（零捏造原则，空值处理，去营销化）
5. 若必填字段缺失，提示客户补充信息，不进入 P08

**输出**：干净的实体字典 JSON，存储至 MySQL。

---

### 场景二：独立站授权模式 - 数据解析（P07）

**触发条件**：用户从独立站授权.html 进入。

**执行逻辑**：
1. 从已同步的站点数据中提取核心字段（网站 title → 企业名称，description → 企业定位，产品页面 → 主营品类）
2. 输出结构化 JSON，格式与文档上传模式统一
3. 后续流程同文档上传模式

**输出**：结构化 JSON，格式与文档模式统一。

---

### 场景三：深度分析与 ICP 建模（P08）

**触发条件**：P07 阶段完成。

**执行逻辑**：
1. 将 P07 输出的实体字典输入 AI 分析 Prompt（Claude 3.5 Sonnet / GPT-4o）
2. AI 综合分析输出：企业定位、业务类型识别、主营品类、核心优势、供应链分析、目标客户画像
3. 根据业务类型生成动态四维评分权重：
   - 外贸型：需求匹配 32% / 购买能力 20% / 采购意向 35% / 活跃意向 13%
   - 生产型：需求匹配 40% / 购买能力 25% / 采购意向 20% / 活跃意向 15%
   - 服务型：需求匹配 35% / 购买能力 25% / 采购意向 15% / 活跃意向 25%
   - 渠道型：需求匹配 35% / 购买能力 20% / 采购意向 30% / 活跃意向 15%
4. 客户信息存储至 MySQL，供后续搜客评分计算使用

**输出**：完整 ICP 画像，存储至 MySQL。

---

### 场景四：处理进度实时展示

**触发条件**：页面加载后异步任务执行中。

**执行逻辑**：
1. 两个阶段并行/串行展示：文本提取与结构化（P07）→ 深度分析与 ICP 建模（P08）
2. 步骤状态切换：等待中（灰色）→ 执行中（蓝色+旋转图标）→ 已完成（绿色+勾选图标）
3. 进度条动画：平滑递增，带波浪流动效果
4. 中心动画：三层同心圆，最内层旋转，外层脉冲扩散
5. 页面组件依次淡入上滑（fadeInUp）
6. 所有步骤完成后 1.5 秒自动跳转至业务分析报告页面

**输出**：实时进度 UI，处理完成自动跳转。

## 边界规则

| 规则 | 条件 | 系统行为 |
|------|------|---------|
| 任务不可中断 | 分析任务开始后 | 不可中断，需等待完成或失败 |
| 超时限制 | 单任务最长执行 | 10 分钟，超时自动标记失败 |
| 并发限制 | 同一租户同时任务数 | 最多 1 个分析任务执行中 |
| 重试限制 | 失败后重试次数 | 最多 3 次，超过需联系客服 |
| 页面关闭 | 用户关闭页面 | 异步任务继续在后台执行 |
| 接口超时 | 单次 API 调用超时 | 1 分钟超时，跳过本次继续执行 |
| P07 必填字段 | 企业名称/企业定位/业务类型/主营品类/目标客户类型/目标客户市场区域缺失 | 中断流程，不进入 P08，提示客户补充 |
| P07 可选字段 | 核心优势/企业规模/供应链分析缺失 | 使用默认值或 AI 推断，不阻断流程 |
| 跳转等待 | 所有步骤完成后 | 1.5 秒后自动跳转至业务分析报告页面 |

## 异常处理

| 异常场景 | 检测方式 | 前端表现 | Toast 文案 |
|---------|---------|---------|-----------|
| 任务不存在 | code 404 | 跳转至初始化页面 | 「任务不存在或已过期，请重新开始」 |
| P07 必填字段缺失 | AI 输出校验 | 中断提示 | 「部分必填字段缺失，请返回补充：{fields}」 |
| P07 文本提取失败 | code 500 | 失败提示 + 重试按钮 | 「文本提取失败，请检查上传的文件」 |
| P08 ICP 建模失败 | code 500 | 失败提示 + 重试按钮 | 「ICP 建模失败，请重试」 |
| 任务超时 | 10 分钟超时 | 超时提示 | 「任务处理超时，请稍后重试」 |
| AI 输出为空 | 校验 null | 自动重试 1 次 | 「AI 分析结果为空，正在重试...」 |
| AI 格式错误 | JSON 解析失败 | 自动重试 1 次 | 「AI 返回数据格式异常，正在自动重试...」 |
| AI 内容幻觉 | 与输入材料矛盾 | 标记异常 | 「AI 分析结果存在异常，建议重新分析」 |
| AI 字段缺失 | 必要字段为空 | 默认值填充 | 「AI 分析不完整，部分数据缺失」 |
| AI 评分异常 | 超出 0-100 范围 | 默认值替代 | 「AI 评分异常，已使用默认值」 |
| 网络超时 | 请求超时 | 错误提示 | 「网络连接异常，请检查网络后重试」 |
| 服务不可用 | code 500 | 全局错误处理 | 「服务暂时不可用，请稍后再试」 |
| 跳转失败 | 页面跳转失败 | 手动点击按钮跳转 | 「页面跳转失败，请手动继续」 |
