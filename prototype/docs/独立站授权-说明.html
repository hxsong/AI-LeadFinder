功能说明文档
P05-P06 节点

独立站授权页面

对应业务流程节点：P05-P06（授权系统自动获取独立站信息 → 自动同步站点产品与企业介绍数据）

页面概述

本页面利用"独立站"底层优势，通过 API 毫秒级获取高纯度、高结构化的客户商业意图数据。用户选择需要授权的独立站后，系统自动同步企业介绍（含目标市场、资质证书）、主营品类、SEO 关键词等结构化信息，为后续 AI 业务分析提供精准数据源。此流程与 P03 互斥。

可见功能逻辑

1. 独立站列表展示

站点卡片信息

  • • 站点名称：如"XX科技官方网站"
  • • 站点域名：如"xx-tech.example.com"
  • • 状态标签：已上线/未上线
选择限制：最多选择 1 个站点作为核心主站

2. 站点选择交互

操作 视觉效果 触发行为
点击站点卡片 橙色边框 + 渐变背景 + 勾选图标 展开数据同步预览区域
悬停站点卡片 上浮 2px + 阴影效果 -
切换选择 原选中取消，新选中高亮 重新同步数据预览

3. 数据同步预览区域

选择站点后，显示数据同步预览区域，包含以下模块：

企业介绍

  • • 公司简介文本
  • • 成立时间、规模等
  • • 目标市场（文本）
  • • 资质证书（文本）

主营品类

  • • 一级分类（标签）
  • • 二级分类（标签）

SEO 关键词

  • • 核心关键词（标签）
  • • 长尾关键词（标签）

同步状态指示

正在同步...
同步完成

4. 摘要卡片

已选择站点

XX科技官方网站

预计分析时长

约 1-2 分钟

5. 提交按钮状态

条件 按钮状态 跳转目标
未选择站点 灰色禁用 -
已选择站点 橙色可点击 AI业务分析中.html?source=site

6. 分支流程

返回上一步

点击"返回上一步"按钮，返回 业务初始化.html 重新选择数据来源方式

服务端处理逻辑：

  • • 调用建站系统内部只读 API 网关
  • • 毫秒级获取高纯度、高结构化的客户商业意图数据
  • • 数据包括：企业介绍（含目标市场、资质证书）、主营品类、SEO 关键词
  • • 无需用户手动输入，自动完成数据采集

3. 确认授权接口

POST /api/v1/sites/{site_id}/authorize

Request Headers:
  Authorization: Bearer {tenant_token}

Request Body:
{
  "site_id": "SITE-001",
  "sync_id": "SYNC-2024-001"
}

Response:
{
  "code": 200,
  "data": {
    "authorization_id": "AUTH-2024-001",
    "site_id": "SITE-001",
    "status": "authorized",
    "next_step": "/processing?source=site"
  }
}

4. 异常流程处理

异常场景 错误码 前端处理
租户无独立站 404 显示空状态，提示"暂无可用独立站"
站点未上线 403 站点卡片置灰，不可选择
数据同步失败 500 显示同步失败提示，支持重试
API 超时 504 显示超时提示，支持重试

限制规则

站点数量限制

最多选择 1 个站点作为核心主站进行授权

状态限制

仅"已上线"状态的站点可选择授权

权限限制

仅租户管理员可执行授权操作

互斥限制

与 P03 上传材料方式互斥，不可同时使用
