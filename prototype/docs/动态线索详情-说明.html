功能说明文档
P46 节点

动态线索详情

对应业务流程节点：P46（展示动态线索分析报告）

页面概述

动态线索详情页是企业身份被揭开后的"真相时刻"。由于底层调用了相同的联系人接口，此页面的展示形态需要与 P27（静态线索报告）高度对齐，但在核心依据上，要用"动态行为"替代"静态海关/展会数据"。页面展示最终动态线索评分、企业身份解析结果、原始动态行为证据、联系人列表、账单透明化和行动召唤按钮。

可见功能逻辑

1. 企业核心信息区

企业基础信息

  • • 企业名称（可点击跳转官网）
  • • 企业域名（超链接）
  • • 所在地
  • • 企业规模（员工数）

2. 联系人列表（与静态线索详情保持一致）

联系人标识

  • • 联系人卡片统一样式
  • • 姓名 + 职位标签
  • • 按重要性排序展示

联系人信息

  • • 头像（姓名首字母）
  • • 姓名 + 职位标签
  • • 邮箱（可点击发送邮件）
  • • 邮箱验证状态徽章
  • • 社媒链接（LinkedIn/Twitter）

操作按钮

  • • 所有联系人：统一按钮样式
  • • 点击后跳转至智能触达 (ITA) 服务

3. 原始动态行为证据（与静态线索的差异点）

复刻 P37 的"微观行为轨迹轴"，作为该线索高热度的铁证

访客基础信息

  • • 访客 IP（脱敏显示）
  • • 地理位置
  • • 设备类型 / 操作系统
  • • 流量来源
  • • 会话时间

基础行为分

  • • 基础行为意向分（如：95分）
  • • 最终动态线索分计算说明

微观行为轨迹轴

  • • 时间线形式展示每个行为事件
  • • 行为类型标签（页面浏览/复制文本/悬停交互/意图流失）
  • • 行为详情（URL、停留时长、滚动深度等）
  • • AI 行为解读（每个行为的意义分析）

关键微观事件摘要

  • • 滚动深度: 85% (阅完参数表)
  • • 复制了文本: 'LD-P1.9-PRO'
  • • 表单弃填: Contact 页面

4. 账单透明化

费用明细（基于 Apollo.io Credits 消耗）

  • • IP 企业解析（IRS 服务）：3 Credits
  • • 基础联系人数据：不消耗 Credits
  • • 邮箱揭示（reveal_personal_emails）：每个有效邮箱 1 Credit
  • • 电话号码揭示（reveal_phone_number）：每个有效电话 1 Credit
  • • LinkedIn 主页附加：包含在基础数据中
  • • 本次消耗总计（加粗高亮）

评分计算说明

最终动态线索分（如：95）

5. 分支流程

A

返回线索列表

点击底部"返回线索列表"按钮，跳转至 动态线索池.html

B

使用智能触达

点击底部主按钮，跳转至智能触达 (ITA) 服务页面，进入触达流程

C

发送邮件

点击联系人卡片的"发送邮件"按钮，跳转至智能触达页面并预选该联系人

6. ISP 宽带线索特殊处理

当 IP 解析结果为公共宽带（ISP）时，页面展示特殊状态

ISP 提示

  • • 企业名称显示"ISP 匿名访客"
  • • 显示 ISP 运营商名称
  • • 警告提示：未能精准定位企业主体

保留数据

  • • 行为意向分保留
  • • AI 判词建议持续监测
  • • 联系人矩阵显示"未找到联系人"
  • • 账单仅显示 IP 解析成本

限制规则

权限控制

当前版本系统暂无权限控制逻辑，所有登录用户均可查看动态线索详情。

数据只读

线索详情页为只读模式，不支持编辑联系人信息

线索状态

点击深度挖掘的线索，无论是否挖掘成功，都会进入线索池，都能查看线索详情

线索池中没有线索评分字段

IP 解析处理逻辑

深度挖掘时调用 IP 解析服务，处理逻辑如下：

  • • 解析成功：返回企业名称、地址、行业等信息，生成动态线索
  • • 解析失败（无法识别）：提示用户「该 IP 无法解析到企业」，不扣除 Credits
  • • 解析失败（服务异常）：提示用户「IP 解析服务暂时不可用」，不扣除 Credits
  • • ISP 宽带线索：页面展示「该 IP 为 ISP 宽带 IP，无法解析到具体企业」提示

AI 综合研判异常检测

AI 模型存在幻觉风险，可能导致综合研判结论与实际情况不符。本模块对 AI 综合研判输出进行专项检测。

  • • 输出为空检测：AI 返回空研判时触发，提示「AI 研判结果为空」
  • • 格式校验：研判 JSON 解析失败时触发，提示「AI 返回数据格式异常」
  • • 一致性校验：研判与原始数据矛盾时触发，提示「AI 研判仅供参考」
  • • 完整性校验：必要研判字段缺失时触发，提示「AI 研判不完整」

AI 综合研判异常处理策略

异常类型 处理方式 提示文案
输出为空 显示默认研判模板 「AI 研判结果为空」
格式错误 显示原始数据摘要 「AI 返回数据格式异常」
内容幻觉 标记仅供参考，提示用户核实 「AI 研判仅供参考，以实际数据为准」
字段缺失 使用默认值填充 「AI 研判不完整」
