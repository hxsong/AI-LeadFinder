## 1. 原型概览

AI-LeadFinder 产品原型包含 **21 个 HTML 原型页面**和 **21 个功能说明文档**，覆盖产品从账号开通到线索管理的完整流程（P01-P80）。

- 原型页面目录：`prototype/pages/`
- 功能说明文档：`prototype/docs/`
- 全局样式：`prototype/styles/global.css`

## 3. 页面跳转关系

### 3.1 账号与资源管理（P01-P11）

```
服务商开通账号（P01）
    ↓
资源下发确认（P02）
    ↓
企业管理员登录 → 多BOT工作台（P03）
    ↓ 选择数据源
├─ 上传业务材料（P05）
│       ↓
│   AI业务分析中（P08）→ 业务分析报告（P09-P11）
│           ↓
└─ 独立站授权（P06-P07）
        ↓
    AI业务分析中（P08）→ 业务分析报告（P09-P11）
```

### 3.2 静态数据获客（P12-P27）

```
业务分析报告 → 搜客需求输入（P12）
    ↓
搜客策略预览（P13-P16）
    ↓
搜客执行中（P16）
    ↓
搜客结果列表（P20-P22）
    ↓ 挖掘联系人
静态线索池（P22-P25）
    ↓
线索详情（P26-P27）
```

### 3.3 动态流量获客（P28-P46）

```
多BOT工作台 → 探针部署（P28）
    ↓
访客雷达（P29-P38）
    ↓ 查看详情
访客行为详情（P39-P42）
    ↓ 深度挖掘
动态线索池（P43-P46）
    ↓
动态线索详情（P47-P70，预留）
```

## 4. 侧边栏导航结构

进入 BOT 后，左侧边栏包含以下模块（按顺序）：

```
├─ 搜索客户（对应 P12-P22）
│   ├─ 搜客需求输入
│   ├─ 搜客策略预览
│   ├─ 搜客执行中
│   ├─ 搜客结果列表
│   └─ 搜客记录
├─ 访客雷达（对应 P28-P42）
│   ├─ 探针部署
│   ├─ 访客雷达
│   └─ 访客行为详情
├─ 静态线索（对应 P22-P27）
│   ├─ 静态线索池
│   └─ 线索详情
└─ 动态线索（对应 P43-P70）
    ├─ 动态线索池
    └─ 动态线索详情
└─ 全局设置（AI Prompt 管理）
    └─ 提示词语言管理
```

## 5. 页面样式规范

- **主色调**：`#3B82F6`（蓝色）
- **字体**：Plus Jakarta Sans（正文）/ Fira Sans（代码）
- **原型展示入口**：左侧浮层导航（entry-nav.js）或顶部导航栏
- **目录面板**：右侧浮层面板（catalog-panel.js）
- **样式规范文件**：`prototype/styles/global.css`

## 6. 原型使用说明

- 所有原型页面可直接用浏览器打开（HTML 文件，无需服务器）
- 功能说明文档用于研发实现参考，每个文档对应一个页面
- 原型与 PRD 的双向索引关系：本文档映射表 + prd-md 各文件头部的"关联原型页面"
