## 数据契约

| 前端展示字段 | 后端字段名 | 类型 | 来源/存储 | 说明 |
|-------------|-----------|------|----------|------|
| BOT ID | `bot_id` | `string` | 路由参数传入 | 当前 BOT 唯一标识 |
| 自然语言需求 | `user_query` | `string` | 用户输入 | 最多 500 字符，超出自动截断 |
| 快速筛选标签 | `filter_tags` | `array` | 用户选择 | 美国/欧洲/有海关记录/近期参展/大型企业 |
| 目标企业数量 | `target_count` | `number` | 用户输入 | 范围 20-200，默认 100 |
| 使用默认 ICP | `use_default_icp` | `boolean` | 系统生成 | user_query 为空时自动设为 true |
| 搜客任务 ID | `task_id` | `string` | 系统生成 | 后端返回的异步任务标识 |
| 任务状态 | `status` | `string` | 接口返回 | pending / running / completed / failed |
| Credits 余额 | `credits_balance` | `number` | 接口返回 | 当前租户 Credits 余额 |
| ICP 画像状态 | `icp_status` | `string` | 接口返回 | 是否存在有效 ICP 画像 |

## 页面流转

### 入口

| 来源 | 触发条件 | 携带参数 |
|------|---------|---------|
| 业务分析报告.html | 用户点击"开始搜客"按钮 | `bot_id` |
| 搜客记录.html | 用户点击"新建搜客"按钮 | `bot_id` |

**入口前置条件**：当前 BOT 已完成 P08 业务分析，存在有效 ICP 画像。

### 出口

| 目标 | 触发条件 | 携带数据 |
|------|---------|---------|
| 搜客策略预览.html | 用户确认需求并提交 | `bot_id`, `task_id` |
| 搜客执行中.html | 进入页面时检测到已有任务在执行 | —（前端自动跳转） |
| 搜客记录.html | 用户主动点击侧边栏"搜客记录" | — |

### 页面状态

| 状态 | 说明 | 进入条件 | 退出条件 |
|------|------|---------|---------|
| 加载中 | 页面初始化，查询任务状态 | 页面加载 | 任务状态查询完成 |
| 有进行中任务 | 检测到当前 BOT 有执行中任务 | 查询到 running 状态任务 | 自动跳转至搜客执行中 |
| 正常编辑 | 无进行中任务，允许输入需求 | 无 running 任务 | 用户提交后跳转 |
| 提交中 | 已点击确认搜客，等待响应 | 用户点击"确认搜客" | 后端返回后跳转 |

**状态流转图**：
```
加载中 --(无running任务)--> 正常编辑 --(点击确认)--> 提交中 --(返回task_id)--> 搜客策略预览.html
加载中 --(有running任务)--> 有进行中任务 --(自动跳转)--> 搜客执行中.html
```

## 数据流

### 提交搜客需求

```
POST /api/v1/search/input
```

**请求头**：`Authorization: Bearer {tenant_token}`
**Content-Type**：`application/json`

**请求参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| tenant_id | `string` | 是 | 租户 ID |
| bot_id | `string` | 是 | BOT 实例 ID |
| user_query | `string` | 否 | 用户输入的自然语言需求；为空时后端使用 ICP 画像 |
| target_count | `number` | 否 | 目标企业数量，默认 100，范围 20-200 |
| use_default_icp | `boolean` | 否 | 是否使用默认 ICP；user_query 为空时自动设为 true |

**响应结构**：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "task_id": "STR-xxx",
    "status": "pending"
  }
}
```

**响应码说明**：

| code | 含义 |
|------|------|
| 200 | 任务创建成功 |
| 400 | 参数错误 |
| 1002 | Credits 余额不足 |
| 500 | 服务异常 |

---

### 查询任务状态

```
GET /api/v1/search/status?bot_id={bot_id}
```

**请求头**：`Authorization: Bearer {tenant_token}`

**响应结构**：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "has_running_task": true,
    "task_id": "TASK-202401-001",
    "status": "running"
  }
}
```

**响应码说明**：

| code | 含义 |
|------|------|
| 200 | 查询成功 |
| 401 | 未登录 |
| 500 | 服务异常 |

## 处理逻辑

### 场景一：正常搜客流程

**触发条件**：用户有明确的短期搜客需求，且 Credits 余额充足。

**执行逻辑**：

1. 系统接收请求：用户输入自然语言需求（最大 500 字符）
2. 系统执行处理：
   - 用户可选点击快速筛选标签，标签内容自动追加到文本框
   - 用户拖动滑块设置目标企业数量（默认 100 家，范围 20-200）
   - 用户点击"确认搜客"按钮
   - 前端将请求发至 `POST /api/v1/search/input`
3. 系统返回响应：后端返回 `task_id`，前端跳转至搜客策略预览页面

**边界条件**：

| 条件 | 系统行为 |
|------|---------|
| user_query 超过 500 字符 | 超出部分自动截断，不报错 |
| Credits 余额 ≤ 0 或余额 < 预估消耗 | "确认搜客"按钮禁用，提示充值 |
| 未完成 P08 业务分析 | 按钮禁用，提示"请先完成业务初始化" |
| 当前 BOT 已有搜客任务执行中 | 进入页面时自动跳转至搜客执行中页面 |
| ICP 数据不存在或已过期 | 禁止提交，提示"请重新生成业务分析报告" |

**状态变更**：

| 原状态 | 新状态 | 触发条件 |
|--------|--------|----------|
| 正常编辑 | 提交中 | 用户点击"确认搜客" |
| 提交中 | 正常编辑 | 接口返回错误 |

**输出**：跳转到搜客策略预览页面，携带 `bot_id` 和 `task_id`。

---

### 场景二：使用 ICP 画像默认搜索

**触发条件**：用户未输入任何文字，直接提交空表单。

**执行逻辑**：

1. 系统接收请求：前端检测 `user_query` 为空
2. 系统执行处理：
   - 自动设置 `use_default_icp=true`
   - 请求发送至 `POST /api/v1/search/input`
   - 后端读取该 BOT 的 P09 ICP 画像作为搜索指令
3. 系统返回响应：返回 `task_id`，前端跳转

**边界条件**：

| 条件 | 系统行为 |
|------|---------|
| ICP 画像不存在 | 返回错误，提示重新生成业务分析报告 |
| ICP 画像已过期 | 返回错误，提示重新分析 |

**状态变更**：

| 原状态 | 新状态 | 触发条件 |
|--------|--------|----------|
| 正常编辑 | 提交中 | 用户提交空表单 |

**输出**：与场景一相同，跳转到搜客策略预览页面。

---

### 场景三：已有任务在执行

**触发条件**：进入页面时，后端检测到该 BOT 存在状态为"执行中"的搜客任务。

**执行逻辑**：

1. 系统接收请求：前端在进入页面时调用 `GET /api/v1/search/status`
2. 系统执行处理：
   - 查询当前 BOT 的任务状态
   - 若存在进行中任务，前端自动跳转到搜客执行中页面
3. 系统返回响应：无表单交互，直接跳转

**边界条件**：

| 条件 | 系统行为 |
|------|---------|
| 查询任务状态失败 | 提示错误，允许用户重试 |

**状态变更**：

| 原状态 | 新状态 | 触发条件 |
|--------|--------|----------|
| 加载中 | 有进行中任务 | 查询到 running 状态任务 |
| 有进行中任务 | 已跳转 | 自动跳转至搜客执行中页面 |

**输出**：自动跳转至搜客执行中页面，无任何表单交互。

## 异常兜底

| 异常场景 | 检测方式 | 后端处理 | 前端表现 |
|---------|---------|---------|---------|
| 网络超时 | 请求无响应超过 30s | 重试 1 次 | Toast「网络连接异常，请检查网络后重试」 |
| LLM 超时（>60s） | gpt-4o 调用超时 | 降级返回预置策略模板 | Toast「AI 分析超时，已使用默认策略」 |
| Credits 不足 | 余额 < 预估消耗 | 不发起请求 | 按钮禁用，Toast「额度不足，请充值」 |
| 任务并发 | search/status 返回有进行中任务 | — | 自动跳转至搜客执行中页面 |
| ICP 数据缺失 | /bot/{id}/icp 返回 404 | — | Toast「ICP 画像不存在，请重新生成」，按钮禁用 |
| 服务不可用 | HTTP 500 | 记录错误日志 | Toast「服务暂时不可用，请稍后再试」 |
| 参数错误 | 接口返回 400 | 返回具体错误字段 | 根据错误字段高亮对应输入框 |
