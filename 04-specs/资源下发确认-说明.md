## 概述

资源下发确认页面用于在资源正式发放前展示下发详情，供管理员确认。该页面作为操作示例参考，实际资源配置和开通逻辑与独立应用一致，参考访客雷达的开通和配置过程。

---

## 页面入口与出口

### 入口

| 来源 | 触发条件 | 携带参数 |
|------|---------|---------|
| 租户资源充值.html | 充值成功后跳转 | `tenant_id`, `transaction_id` |

### 出口

| 目标 | 触发条件 | 携带数据 |
|------|---------|---------|
| 多BOT工作台.html | 点击"确认生效"按钮 | `tenant_id` |
| 租户资源充值.html | 点击"返回修改"或"取消下发" | `tenant_id` |

---

## 功能逻辑

### 场景一：查看资源下发详情

**触发条件**：从充值页面跳转进入。

**执行逻辑**：
1. 页面展示本次下发的资源详情：
   - 租户名称、Tenant ID、交易流水号、当前系统到期时间
   - 系统服务续费：新增年数、当前 BOT 席位数、单价、费用、续费后到期时间
   - BOT 席位增购：增购数量、当前服务剩余年数、单价、费用
   - Credits 资源包：包类型、Credits 数量、折扣、费用
2. 展示下发状态进度（下发中 → 已确认 → 已生效）
3. 管理员确认信息无误后，点击"确认生效"按钮

**输出**：资源下发确认状态。

---

### 场景二：确认资源生效

**触发条件**：管理员确认资源信息无误。

**执行逻辑**：
1. 前端发送确认请求
2. 后端校验交易状态
3. 执行资源生效：
   - 系统服务：更新到期时间
   - BOT 席位：增加可用席位
   - Credits：增加租户余额
4. 返回成功结果，前端跳转工作台

**输出**：资源生效提示，页面跳转回工作台。

---

### 场景三：取消下发

**触发条件**：管理员发现信息有误。

**执行逻辑**：
1. 点击"取消下发"按钮
2. 后端撤销本次下发记录，不扣除任何费用
3. 页面跳转回充值页面

**输出**：撤销成功提示。

---

### 场景四：返回修改

**触发条件**：管理员需要修改资源内容。

**执行逻辑**：
1. 点击"返回修改"按钮
2. 页面跳转回充值页面，保留已选资源状态

**输出**：返回充值页面。

## 页面状态

| 状态 | 触发条件 | UI 表现 |
|-----|---------|--------|

## 边界规则

| 规则 | 条件 | 系统行为 |
|------|------|---------|
| 撤销操作 | 确认前取消 | 撤销下发，不扣除任何费用 |
| 重复确认 | 重复点击确认按钮 | 按钮禁用，防止重复确认 |
| 超时返回 | 30 分钟无操作 | 提示返回工作台 |
| 已生效交易 | 状态为 active | 提示无需重复确认 |
| 已取消交易 | 状态为 cancelled | 提示交易已取消 |

## API 清单

| 接口 | 方法 | 说明 | 调用时机 |
|------|------|------|---------|
| /internal/v1/tenant/resources/detail | GET | 获取下发详情 | 页面加载 |
| /internal/v1/tenant/resources/confirm | POST | 确认资源生效 | 点击确认生效按钮 |
| /internal/v1/tenant/resources/cancel | POST | 取消下发 | 点击取消下发按钮 |

## 异常处理

| 异常场景 | 检测方式 | 前端表现 | Toast 文案 |
|---------|---------|---------|-----------|
| 资源已生效 | 状态校验 | 提示并跳转 | 「该资源已生效，无需重复确认」 |
| 资源不存在 | code 404 | 错误提示 | 「该下发记录不存在或已过期」 |
| 网络超时 | 请求超时 | 错误提示 | 「网络连接异常，请检查网络后重试」 |
| 服务不可用 | code 500 | 全局错误处理 | 「服务暂时不可用，请稍后再试」 |
| 交易已取消 | 状态校验 | 提示并跳转 | 「该交易已取消，请重新下单」 |

## 字段映射

| 前端展示字段 | 后端字段名 | 数据来源说明 |
|-------------|-----------|------------|
| 租户名称 | tenant_name | 租户注册信息 |
| Tenant ID | tenant_id | 系统生成 |
| 交易流水号 | transaction_id | 系统生成 |

| 系统服务费 | resources.service_fee.fee | 后端计算 |
| 续费后到期时间 | resources.service_fee.new_expiry_date | 后端计算 |

| Credits 折扣 | resources.credits_packages[].discount | 配置数据 |
| Credits 包价格 | resources.credits_packages[].price | 充值请求数据 |
| 费用汇总 | fee_summary | 后端计算 |
| 下发状态 | status | pending / active / cancelled |
| 生效时间 | activated_at | 确认后系统时间 |
| 取消时间 | cancelled_at | 取消后系统时间 |
