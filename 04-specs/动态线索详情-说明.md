## 概述

动态线索详情页是企业身份被揭开后的"真相时刻"。底层调用与静态线索相同的联系人接口，展示形态需与静态线索详情（P27）高度对齐，但核心证据用"动态行为"替代"静态海关/展会数据"。页面展示最终动态线索评分、企业身份解析结果、原始动态行为证据、联系人列表、账单透明化和行动召唤按钮。

## 功能逻辑

### 场景一：正常查看动态线索详情

**触发条件**：用户从动态线索池点击某条已完成线索卡片进入。

**执行逻辑**：
1. 前端请求 `GET /api/v1/dynamic-leads/{dynamic_lead_id}` 获取完整报告
2. 页面展示企业核心信息区（名称、域名、所在地、规模）
3. 展示最终动态线索评分与 AI 判词
4. 展示原始动态行为证据（访客基础信息 + 微观行为轨迹轴 + 关键微观事件摘要）
5. 展示联系人矩阵（姓名、职位、邮箱、验证状态、社媒链接）
6. 展示账单透明化摘要
7. 展示联络人操作按钮（发送邮件 / 跳转 ITA）

**输出**：完整动态线索报告页面。

---

### 场景二：ISP 宽带线索特殊展示

**触发条件**：IP 解析结果为公共宽带（ISP），无法精准定位企业主体。

**执行逻辑**：
1. 企业名称区域显示"ISP 匿名访客"
2. 显示 ISP 运营商名称（如 Comcast）
3. 警告提示"未能精准定位企业主体"
4. 保留行为意向分和 AI 判词建议持续监测
5. 联系人矩阵显示"未找到联系人"
6. 账单仅显示 IP 解析成本

**输出**：ISP 宽带线索特殊展示页面。

---

### 场景三：AI 判词异常

**触发条件**：AI 模型返回空内容、格式错误、内容幻觉或字段缺失。

**执行逻辑**：
1. 前端检测 AI 判词字段异常状态
2. 按异常类型显示对应降级文案（详见异常处理章节）
3. 页面其余内容正常展示

**输出**：线索报告页面，AI 判词区域显示降级文案。

## 页面状态

| 状态 | 触发条件 | UI 表现 |
|-----|---------|---------|
| 正常 | 数据完整返回 | 展示完整报告 |
| 加载中 | 请求发出后 | 骨架屏或 loading 动画 |
| ISP 线索 | IP 解析为宽带 | ISP 警告标签 + 匿名企业展示 |
| AI 已触发 | 规则打分 ≥ 60 | 右侧面板显示 AI 洞察和综合研判 |
| AI 未触发 | 规则打分 < 60 | 不显示 AI 洞察区域 |
| 无联系人 | ISP 解析失败或无联系人 | 联系人矩阵显示空态 |
| AI 异常 | AI 判词字段异常 | AI 判词区域显示降级文案 |

## 边界规则

| 规则 | 条件 | 系统行为 |
|------|------|---------|
| 权限控制 | 当前版本无权限隔离 | 所有登录用户均可查看动态线索详情 |
| 数据只读 | 页面为只读模式 | 不支持编辑联系人信息 |
| 线索状态 | 无论挖掘是否成功 | 点击深度挖掘的线索均能查看线索详情 |
| 线索池字段 | 线索池列表页 | 不显示评分字段 |
| IP 解析成功 | 成功返回企业信息 | 正常展示企业信息、联系人、账单 |
| IP 解析失败 | 无法识别企业 | ISP 线索特殊展示，账单仅显示 IP 解析成本 |
| Credits 扣费 | 解析失败（无法识别 / 服务异常） | 不扣除 Credits |

## 异常处理

| 异常场景 | 检测方式 | 前端表现 | Toast 文案 |
|---------|---------|---------|-----------|
| 线索不存在 | code 404 | 显示"该线索不存在或已被删除"提示，跳转回线索池 | 「该线索不存在或已被删除」 |
| 网络超时 | 请求超时 30s | 显示错误提示 | 「网络连接异常，请检查网络后重试」 |
| 服务不可用 | code 500 | 全局错误处理 | 「服务暂时不可用，请稍后再试」 |
| AI 判词为空 | ai_reasoning 为空 | 显示默认研判模板 | 「AI 研判结果为空」 |
| AI 格式错误 | JSON 解析失败 | 显示原始数据摘要 | 「AI 返回数据格式异常」 |
| AI 内容幻觉 | 与原始数据矛盾 | 标记仅供参考 | 「AI 研判仅供参考，以实际数据为准」 |
| AI 字段缺失 | 必要字段为空 | 使用默认值填充 | 「AI 研判不完整」 |
| 邮箱无效 | 点击无效邮箱 | 提示邮箱地址无效 | 「该联系人的邮箱地址无效」 |
| 邮箱未验证 | 发送未验证邮箱 | 提示发送可能失败 | 「该联系人邮箱未经验证，发送可能失败」 |
| 浏览器限制 | 复制操作不支持 | 提示手动选择复制 | 「浏览器不支持自动复制，请手动选择复制」 |
| 剪贴板权限被拒 | Clipboard API 权限不足 | 提示手动复制 | 「剪贴板访问权限被拒绝，请在浏览器设置中允许」 |

## 字段映射

| 前端展示字段 | 后端字段名 | 数据来源说明 |
|-------------|-----------|------------|
| 动态线索 ID | dynamic_lead_id | 系统生成，UUID |
| 解析是否成功 | resolution_status.is_successful | boolean |
| 企业名称 | resolution_status.company_name | IRS IP 解析服务返回（成功时） |
| 企业域名 | resolution_status.company_domain | IRS IP 解析服务返回（成功时） |
| 行业类型 | resolution_status.industry | IRS IP 解析服务返回（成功时） |
| 最终动态评分 | lead_evaluation.final_dynamic_score | 行为分 + ICP 匹配分综合计算 |
| 线索等级 | lead_evaluation.lead_grade | S/A/B/C，由评分映射 |
| AI 判词 | lead_evaluation.ai_reasoning | LLM 生成，需做空值和格式校验 |
| 基础行为评分 | original_behavior_evidence.base_behavior_score | 打分引擎计算 |
| 访客 IP（脱敏） | original_behavior_evidence.visitor_info.ip_address | GeoIP 脱敏处理 |
| 地理位置 | original_behavior_evidence.visitor_info.geo_location | GeoIP 定位数据 |
| 设备信息 | original_behavior_evidence.visitor_info.device_type | 探针 JS 采集 UA 解析 |
| 流量来源 | original_behavior_evidence.visitor_info.traffic_source | 流量归因数据 |
| 会话时间段 | original_behavior_evidence.visitor_info.session_time | 时间范围字符串 |
| 关键微观事件 | original_behavior_evidence.key_micro_events | 行为事件摘要列表 |
| 行为时间轴 | original_behavior_evidence.behavior_timeline | 行为事件列表含 AI 解读 |
| 行为时间 | behavior_timeline[].time | 时间字符串 |
| 行为类型 | behavior_timeline[].type | page_view / copy_text / file_download / click / form_abandon |
| 行为详情 | behavior_timeline[].details | 行为具体描述 |
| AI 行为解读 | behavior_timeline[].ai_interpretation | LLM 生成 |
| IP 解析成本 | billing_receipt.ip_resolution_cost | IRS 服务消耗，3 Credits |
| 联系人数量 | billing_receipt.contacts_found | Apollo.io 返回 |
| 揭示邮箱数 | billing_receipt.emails_revealed | Apollo.io reveal 结果 |
| 揭示电话数 | billing_receipt.phones_revealed | Apollo.io reveal 结果 |
| 联系人成本 | billing_receipt.contacts_cost | Apollo.io Credits 消耗 |
| 总消耗 | billing_receipt.total_credits_consumed | IP 解析 + 联系人挖掘合计 |
| 联系人姓名 | contacts_matrix[].name | Apollo.io 返回 |
| 联系人职位 | contacts_matrix[].title | Apollo.io 返回 |
| 联系人邮箱 | contacts_matrix[].email | Apollo.io reveal 揭示结果 |
| 邮箱验证状态 | contacts_matrix[].email_verified | Apollo.io 返回 |
| 社媒链接 | contacts_matrix[].social_links | Apollo.io 返回 |
| 入库时间 | created_at | ISO 8601 时间戳 |
