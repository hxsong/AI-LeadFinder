## 概述

探针部署是动态流量捕捉模块的入口页面，负责向企业管理员提供域名绑定与 JS 探针代码的部署功能。系统采用「域名绑定优先」机制，探针代码针对绑定域名生成，部署时校验域名一致性，确保数据采集的准确性与安全性。

## 功能逻辑

### 场景一：OAuth 授权独立站一键安装（分支 A）

**触发条件**：账号已存在已授权独立站（通过 P05 独立站授权流程完成）。

**执行逻辑**：
1. 系统检测已授权独立站，自动提取域名并完成绑定，生成专属探针 ID
2. 页面显示"一键安装追踪探针"主按钮
3. 用户点击按钮，弹出部署进度弹窗，实时展示：连接独立站 API → 注入追踪代码 → 校验域名一致性 → 验证部署状态
4. 部署完成后，弹窗自动关闭，页面底部显示成功卡片"数据已回传，探针已生效"

**输出**：部署成功，探针生效。

---

### 场景二：手动填写域名手动埋点（分支 B）

**触发条件**：账号无已授权独立站，需手动输入域名。

**执行逻辑**：
1. EBA 手动输入网站域名，系统验证域名格式（正则校验）与可访问性（HTTP 请求）后完成绑定
2. 系统生成专属探针 ID，展示 JS 探针代码片段，右上角提供"复制代码"按钮
3. 页面以卡片形式展示三步操作指引：① 复制上方代码 → ② 粘贴到网站 `<head>` 标签 → ③ 等待系统验证生效
4. 用户点击"验证代码是否生效"按钮，弹出验证进度弹窗，检测代码是否正确嵌入并校验请求来源域名与绑定域名一致性

**输出**：域名绑定成功 + 探针代码可复制部署。

---

### 场景三：域名已被其他账号绑定

**触发条件**：用户输入的域名已被其他企业账号绑定。

**执行逻辑**：
1. 后端拒绝绑定请求
2. 前端显示"该域名已被其他账号绑定"提示

**输出**：绑定失败，提示联系客服。

---

### 场景四：账号已有绑定域名

**触发条件**：当前企业账号已存在绑定域名。

**执行逻辑**：
1. 阻止新域名绑定请求
2. 前端显示"当前账号已绑定域名 xxx.com，如需更换请联系管理员"提示

**输出**：阻止新绑定，提示联系管理员。

| 状态 | 触发条件 | UI 表现 |
|-----|---------|---------|

## 数据流

### 请求一：域名绑定

```
POST /api/v1/domain/bind
```

**请求头**：`Authorization: Bearer {enterprise_token}`

**请求体**：

```json
{
  "domain": "example.com",
  "bind_type": "manual"
}
```

**响应体**：

```json
{
  "code": 200,
  "data": {
    "domain": "example.com",
    "probe_id": "LF-X7Y8Z9",
    "bind_type": "manual",
    "bind_at": "2024-02-26T10:00:00Z",
    "status": "bound"
  }
}
```

### 请求二：获取探针代码

```
GET /api/v1/probe/code
```

**请求头**：`Authorization: Bearer {enterprise_token}`

**响应体**：

```json
{
  "code": 200,
  "data": {
    "probe_id": "LF-A1B2C3",
    "bound_domain": "xx-tech.example.com",
    "script_url": "https://cdn.leadfinder.io/probe/LF-A1B2C3.js",
    "embed_code": "<script>(function(w,d,s,l,i){...})(window,document,'script','lfDataLayer','LF-A1B2C3');</script>",
    "created_at": "2024-02-26T10:00:00Z"
  }
}
```

### 请求三：一键部署（已授权站点）

```
POST /api/v1/probe/deploy
```

**请求头**：`Authorization: Bearer {enterprise_token}`

**请求体**：

```json
{
  "site_id": "site_12345",
  "probe_id": "LF-A1B2C3",
  "deploy_type": "auto"
}
```

**响应体**：

```json
{
  "code": 200,
  "data": {
    "deploy_id": "deploy_67890",
    "status": "success",
    "deployed_domain": "xx-tech.example.com",
    "domain_verified": true,
    "deployed_at": "2024-02-26T10:05:00Z",
    "message": "探针已成功部署，域名校验通过"
  }
}
```

### 请求四：验证代码嵌入

```
POST /api/v1/probe/verify
```

**请求头**：`Authorization: Bearer {enterprise_token}`

**请求体**：

```json
{
  "probe_id": "LF-A1B2C3"
}
```

**响应体**：

```json
{
  "code": 200,
  "data": {
    "verified": true,
    "detected_script": "https://cdn.leadfinder.io/probe/LF-A1B2C3.js",
    "source_domain": "xx-tech.example.com",
    "bound_domain": "xx-tech.example.com",
    "domain_match": true,
    "verified_at": "2024-02-26T10:08:00Z",
    "status": "active",
    "message": "探针代码检测成功，域名校验通过，数据采集链路已连通"
  }
}
```

| code | 含义 |
|------|------|
| 200 | 操作成功 |
| 400 | 域名格式无效 |
| 409 | 域名已被其他账号绑定 / 账号已有绑定域名 |
| 500 | 服务异常 |

## 边界规则

| 规则 | 条件 | 系统行为 |
|------|------|---------|
| 一账号一域名 | 每个企业账号仅能绑定一个网站域名 | 不可同时绑定多个站点 |
| 域名唯一性 | 同一域名仅能被一个账号绑定 | 已被绑定则拒绝 |
| 部署域名校验 | 探针部署至非绑定域名 | 拒绝数据上报，返回域名不一致错误 |
| 验证冷却 | 手动验证按钮 30 秒内重复点击 | 返回"验证操作过于频繁"提示 |
| 域名变更 | 需更换已绑定域名 | 联系平台管理员重置 |
| GDPR 合规 | 探针绝对不采集 input 输入内容 | 仅记录"发生过输入行为"，需用户 Cookie 同意后触发 |

## API 清单

| 接口 | 方法 | 说明 | 调用时机 |
|------|------|------|---------|

## 异常处理

| 异常场景 | 检测方式 | 前端表现 | Toast 文案 |
|---------|---------|---------|-----------|
| 域名格式无效 | 正则校验失败 | 绑定按钮禁用 | 「请输入有效的网站域名，格式如 example.com」 |
| 域名已被绑定 | HTTP 409 | 拒绝绑定请求 | 「该域名已被其他账号绑定，如需协助请联系客服」 |
| 账号已有绑定 | 检测到已绑定域名 | 阻止新绑定 | 「当前账号已绑定域名 {domain}，如需更换请联系管理员」 |
| 独立站连接失败 | API 调用失败 | 部署进度弹窗显示失败 | 「无法连接独立站 API，请检查授权是否有效」 |
| 代码注入失败 | 注入返回错误 | 显示失败提示 | 「代码注入失败，请手动复制代码进行部署」 |
| 域名不一致 | domain_match false | 拒绝数据上报 | 「检测到部署域名与绑定域名不一致，请检查部署位置」 |
| 未检测到代码 | 验证返回 verified=false | 失败提示 | 「未检测到探针代码，请确认代码已正确嵌入 head 标签」 |
| 域名校验失败 | domain_match false | 失败提示 | 「域名校验失败，探针部署域名与绑定域名不一致」 |
| 验证冷却中 | 30 秒内重复点击 | 阻止验证 | 「验证操作过于频繁，请 {seconds} 秒后再试」 |
| 复制失败 | 浏览器不支持 Clipboard API | 提示手动复制 | 「当前浏览器不支持自动复制，请手动选择代码」 |
| 网络错误 | 请求失败 | 全局错误处理 | 「网络连接异常，请检查网络后重试」 |
| 服务不可用 | HTTP 500 | 全局错误处理 | 「服务暂时不可用，请稍后再试」 |

## 字段映射

| 前端展示字段 | 后端字段名 | 数据来源说明 |
|-------------|-----------|------------|
| 绑定域名 | domain | 用户输入或 OAuth 自动提取 |
| 探针 ID | probe_id | 系统生成，唯一标识符 |
| 绑定类型 | bind_type | manual（手动）/ oauth（授权） |
| 绑定时间 | bind_at | ISO 8601 时间戳 |
| 脚本 URL | script_url | CDN 地址，含探针 ID |
| 嵌入代码 | embed_code | 完整 `<script>` 标签代码 |
| 部署域名 | deployed_domain | 一键部署时独立站实际域名 |
| 域名匹配 | domain_match | boolean，部署域名与绑定域名是否一致 |
| 部署状态 | status | active / pending / failed |
| 验证状态 | verified | boolean，代码是否被检测到 |
| 请求来源域名 | source_domain | 探针请求时携带的来源域名 |
