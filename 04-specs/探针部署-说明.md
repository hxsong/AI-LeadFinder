## 数据契约

| 前端展示字段 | 后端字段名 | 类型 | 来源/存储 | 说明 |
|-------------|-----------|------|----------|------|
| BOT ID | `bot_id` | `string` | 路由参数传入 | 当前 BOT 唯一标识 |
| 绑定域名 | `domain` | `string` | 用户输入或 OAuth 自动提取 | — |
| 探针 ID | `probe_id` | `string` | 系统生成 | 唯一标识符 |
| 绑定类型 | `bind_type` | `string` | 系统生成 | manual（手动）/ oauth（授权） |
| 绑定时间 | `bind_at` | `string` | 系统生成 | ISO 8601 时间戳 |
| 脚本 URL | `script_url` | `string` | 系统生成 | CDN 地址，含探针 ID |
| 嵌入代码 | `embed_code` | `string` | 系统生成 | 完整 `<script>` 标签代码 |
| 部署域名 | `deployed_domain` | `string` | 接口返回 | 一键部署时独立站实际域名 |
| 域名匹配 | `domain_match` | `boolean` | 接口返回 | 部署域名与绑定域名是否一致 |
| 部署状态 | `status` | `string` | 接口返回 | active / pending / failed |
| 验证状态 | `verified` | `boolean` | 接口返回 | 代码是否被检测到 |
| 请求来源域名 | `source_domain` | `string` | 接口返回 | 探针请求时携带的来源域名 |

## 页面流转

### 入口

| 来源 | 触发条件 | 携带参数 |
|------|---------|---------|
| 多BOT工作台.html | 用户点击侧边栏"探针部署" | `bot_id` |

### 出口

| 目标 | 触发条件 | 携带数据 |
|------|---------|---------|
| 访客雷达.html | 部署成功后点击"跳转访客雷达" | `bot_id` |
| 独立站授权.html | 用户选择 OAuth 方式绑定 | — |

### 页面状态

| 状态 | 说明 | 进入条件 | 退出条件 |
|------|------|---------|---------|
| 有已授权站点 | 检测到已授权独立站 | 系统检测到已授权独立站 | 用户操作 |
| 无已授权站点 | 未检测到授权独立站 | 未检测到授权独立站 | 用户操作 |
| 绑定中 | 域名验证请求发出 | 用户点击绑定按钮 | 绑定接口返回 |
| 部署中 | 一键安装请求发出 | 用户点击一键安装按钮 | 部署接口返回 |
| 部署成功 | 部署完成且验证通过 | 部署接口返回成功 | 用户操作 |
| 部署失败 | 部署失败或域名不一致 | 部署接口返回失败 | 用户操作 |

## 数据流

### 域名绑定

```
POST /api/v1/domain/bind
```

**请求头**：`Authorization: Bearer {enterprise_token}`
**Content-Type**：`application/json`

**请求参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| domain | `string` | 是 | 网站域名 |
| bind_type | `string` | 是 | manual / oauth |

**响应结构**：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "domain": "example.com",
    "probe_id": "LF-X7Y8Z9",
    "bind_type": "manual",
    "bind_at": "2024-02-26T10:00:00Z",
    "status": "bound"
  }
}
```

**响应码说明**：

| code | 含义 |
|------|------|
| 200 | 绑定成功 |
| 400 | 域名格式无效 |
| 409 | 域名已被其他账号绑定 / 账号已有绑定域名 |
| 500 | 服务异常 |

## 处理逻辑

### 场景一：OAuth 授权独立站一键安装

**触发条件**：账号已存在已授权独立站（通过 P05 独立站授权流程完成）。

**执行逻辑**：

1. 系统接收请求：页面加载，检测已授权独立站
2. 系统执行处理：
   - 系统检测已授权独立站，自动提取域名并完成绑定，生成专属探针 ID
   - 页面显示"一键安装追踪探针"主按钮
   - 用户点击按钮，弹出部署进度弹窗，实时展示：连接独立站 API → 注入追踪代码 → 校验域名一致性 → 验证部署状态
   - 部署完成后，弹窗自动关闭，页面底部显示成功卡片"数据已回传，探针已生效"
3. 系统返回响应：返回部署结果

**边界条件**：

| 条件 | 系统行为 |
|------|---------|
| 独立站连接失败 | 部署进度弹窗显示失败，提示"无法连接独立站 API，请检查授权是否有效" |
| 代码注入失败 | 显示失败提示，提示"代码注入失败，请手动复制代码进行部署" |
| 域名不一致 | 拒绝数据上报，提示"检测到部署域名与绑定域名不一致，请检查部署位置" |

**状态变更**：

| 原状态 | 新状态 | 触发条件 |
|--------|--------|----------|
| 有已授权站点 | 部署中 | 用户点击一键安装按钮 |
| 部署中 | 部署成功 | 部署接口返回成功 |
| 部署中 | 部署失败 | 部署接口返回失败 |

**输出**：部署成功，探针生效。

---

### 场景二：手动填写域名手动埋点

**触发条件**：账号无已授权独立站，需手动输入域名。

**执行逻辑**：

1. 系统接收请求：用户手动输入网站域名
2. 系统执行处理：
   - EBA 手动输入网站域名，系统验证域名格式（正则校验）与可访问性（HTTP 请求）后完成绑定
   - 系统生成专属探针 ID，展示 JS 探针代码片段，右上角提供"复制代码"按钮
   - 页面以卡片形式展示三步操作指引：① 复制上方代码 → ② 粘贴到网站 `<head>` 标签 → ③ 等待系统验证生效
   - 用户点击"验证代码是否生效"按钮，弹出验证进度弹窗，检测代码是否正确嵌入并校验请求来源域名与绑定域名一致性
3. 系统返回响应：返回验证结果

**边界条件**：

| 条件 | 系统行为 |
|------|---------|
| 域名格式无效 | 绑定按钮禁用，提示"请输入有效的网站域名，格式如 example.com" |
| 域名已被其他账号绑定 | 拒绝绑定请求，提示"该域名已被其他账号绑定，如需协助请联系客服" |
| 账号已有绑定域名 | 阻止新绑定，提示"当前账号已绑定域名 {domain}，如需更换请联系管理员" |
| 验证冷却 | 手动验证按钮 30 秒内重复点击，返回"验证操作过于频繁"提示 |

**状态变更**：

| 原状态 | 新状态 | 触发条件 |
|--------|--------|----------|
| 无已授权站点 | 绑定中 | 用户点击绑定按钮 |
| 绑定中 | 无已授权站点 | 绑定接口返回失败 |
| 无已授权站点 | 部署成功 | 验证接口返回成功 |

**输出**：域名绑定成功 + 探针代码可复制部署。

---

### 场景三：域名已被其他账号绑定

**触发条件**：用户输入的域名已被其他企业账号绑定。

**执行逻辑**：

1. 系统接收请求：用户输入域名
2. 系统执行处理：
   - 后端拒绝绑定请求
   - 前端显示"该域名已被其他账号绑定"提示
3. 系统返回响应：返回 409 错误

**边界条件**：

| 条件 | 系统行为 |
|------|---------|
| 域名已被绑定 | 拒绝绑定请求 |

**状态变更**：无

**输出**：绑定失败，提示联系客服。

---

### 场景四：账号已有绑定域名

**触发条件**：当前企业账号已存在绑定域名。

**执行逻辑**：

1. 系统接收请求：用户尝试绑定新域名
2. 系统执行处理：
   - 阻止新域名绑定请求
   - 前端显示"当前账号已绑定域名 xxx.com，如需更换请联系管理员"提示
3. 系统返回响应：返回 409 错误

**边界条件**：

| 条件 | 系统行为 |
|------|---------|
| 账号已有绑定域名 | 阻止新绑定 |

**状态变更**：无

**输出**：阻止新绑定，提示联系管理员。

## 异常兜底

| 异常场景 | 检测方式 | 后端处理 | 前端表现 |
|---------|---------|---------|---------|
| 域名格式无效 | 正则校验失败 | 拒绝绑定请求 | 绑定按钮禁用，Toast「请输入有效的网站域名，格式如 example.com」 |
| 域名已被绑定 | HTTP 409 | 拒绝绑定请求 | Toast「该域名已被其他账号绑定，如需协助请联系客服」 |
| 账号已有绑定 | 检测到已绑定域名 | 阻止新绑定 | Toast「当前账号已绑定域名 {domain}，如需更换请联系管理员」 |
| 独立站连接失败 | API 调用失败 | 记录错误日志 | 部署进度弹窗显示失败，Toast「无法连接独立站 API，请检查授权是否有效」 |
| 代码注入失败 | 注入返回错误 | 记录错误日志 | 显示失败提示，Toast「代码注入失败，请手动复制代码进行部署」 |
| 域名不一致 | domain_match false | 拒绝数据上报 | Toast「检测到部署域名与绑定域名不一致，请检查部署位置」 |
| 未检测到代码 | 验证返回 verified=false | — | 失败提示，Toast「未检测到探针代码，请确认代码已正确嵌入 head 标签」 |
| 域名校验失败 | domain_match false | — | 失败提示，Toast「域名校验失败，探针部署域名与绑定域名不一致」 |
| 验证冷却中 | 30 秒内重复点击 | 阻止验证 | Toast「验证操作过于频繁，请 {seconds} 秒后再试」 |
| 复制失败 | 浏览器不支持 Clipboard API | — | 提示手动复制，Toast「当前浏览器不支持自动复制，请手动选择代码」 |
| 网络错误 | 请求失败 | 重试 1 次 | Toast「网络连接异常，请检查网络后重试」 |
| 服务不可用 | HTTP 500 | 记录错误日志 | Toast「服务暂时不可用，请稍后再试」 |
