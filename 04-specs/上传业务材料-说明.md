## 概述

上传业务材料页面用于接收用户本地的非结构化业务凭证（PDF/WORD/MD/TXT），作为 AI 画像分析的物理语料。用户通过点击上传或拖拽方式提交文件，系统验证格式和大小后上传至存储，后端触发 AI 分析流程。

## 功能逻辑

### 场景一：文件上传（点击方式）

**触发条件**：用户点击上传区域触发文件选择器。

**执行逻辑**：
1. 点击上传区域，打开系统文件选择器
2. 选中文件后验证格式（PDF/WORD/MD/TXT）和大小（单文件最大 5MB）
3. 验证通过后添加到上传队列
4. 上传完成后显示文件列表（文件名、大小、状态）

**输出**：文件进入上传队列。

---

### 场景二：文件上传（拖拽方式）

**触发条件**：用户将文件拖拽至上传区域。

**执行逻辑**：
1. 监听 dragover 事件，显示蓝色边框 + 渐变背景 dragover 状态
2. 放开文件后自动触发文件选择
3. 后续流程同点击方式

**输出**：拖拽文件进入上传队列。

---

### 场景三：多文件上传

**触发条件**：用户选择多个文件。

**执行逻辑**：
1. 支持一次选择多个文件，逐个添加到文件列表
2. 每个文件独立验证格式和大小
3. 最多上传 3 个文件
4. 上传进度独立显示

**输出**：多个文件并行上传。

---

### 场景四：文件上传提交

**触发条件**：用户点击"开始分析"按钮。

**执行逻辑**：
1. 校验至少上传了 1 个文件
2. 校验所有文件均已上传完成
3. 调用文件上传接口，传递 file_ids
4. 后端存储文件元数据至数据库，返回存储 URL
5. 触发 AI 文本提取任务
6. 跳转到 AI 业务分析中页面

**输出**：文件上传完成，跳转至分析处理页。

## 数据流

### 请求：上传业务材料

```
POST /api/v1/materials/upload
```

**请求头**：`Authorization: Bearer {tenant_token}`
**Content-Type**：`multipart/form-data`

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | File | 是 | 上传文件（PDF/WORD/MD/TXT） |
| bot_id | string | 是 | BOT ID |

**响应体**：

```json
{
  "code": 200,
  "data": {
    "material_id": "MAT-202401-001",
    "file_name": "产品手册.pdf",
    "file_size": 2097152,
    "file_type": "application/pdf",
    "storage_url": "https://cdn.example.com/materials/MAT-202401-001.pdf",
    "upload_status": "completed"
  }
}
```

| code | 含义 |
|------|------|
| 200 | 上传成功 |
| 400 | 文件格式不支持或超出大小限制 |
| 401 | 未登录 |
| 500 | 服务异常 |

## 边界规则

| 规则 | 条件 | 系统行为 |
|------|------|---------|
| 文件格式 | 仅支持 PDF/WORD/MD/TXT | 其他格式返回 400 错误 |
| 单文件大小 | 最大 5MB | 超出返回 400 错误 |
| 文件数量 | 最多 3 个 | 第 4 个文件无法添加 |
| 存储位置 | 上传至 AWS S3 / 阿里云 OSS | 后端自动选择 |
| 文件名 | 不做限制 | 支持中英文文件名 |
| 重复文件 | 同名文件 | 允许上传，系统生成唯一 material_id |

## 异常处理

| 异常场景 | 检测方式 | 前端表现 | Toast 文案 |
|---------|---------|---------|-----------|
| 文件格式不支持 | 文件类型校验 | 错误提示 | 「不支持的文件格式，请上传 PDF/WORD/MD/TXT」 |
| 文件超过 5MB | 文件大小校验 | 错误提示 | 「文件大小超出限制，最大支持 5MB」 |
| 上传数量已达上限 | 文件数量校验 | 上传区域禁用 | 「已上传 3 个文件，达上传上限」 |
| 网络超时 | 请求超时 | 错误提示 | 「网络连接异常，请检查网络后重试」 |
| 服务不可用 | code 500 | 全局错误处理 | 「服务暂时不可用，请稍后再试」 |
| 未登录 | code 401 | 跳转登录页 | 「请先登录」 |

## 字段映射

| 前端展示字段 | 后端字段名 | 数据来源说明 |
|-------------|-----------|------------|
| 文件名称 | file_name | 用户选择文件时读取 |
| 文件大小 | file_size | 前端字节数 |
| 文件类型 | file_type | MIME 类型 |
| 存储 URL | storage_url | OSS/S3 返回地址 |
| 上传状态 | upload_status | pending / uploading / completed / failed |

| BOT ID | bot_id | 路由参数传入 |
