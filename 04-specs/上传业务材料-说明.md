## 概述

上传业务材料页面用于接收用户本地的非结构化业务凭证（PDF/WORD/MD/TXT），作为 AI 画像分析的物理语料。用户通过点击上传或拖拽方式提交文件，系统验证格式和大小后上传至存储，后端触发 AI 分析流程。

---

| BOT ID | bot_id | string | 路由参数传入 | 路由参数传入 |

## 页面流转

### 入口定义

| 来源页面 | 触发条件 | 携带参数 |
|---------|---------|---------|
| 业务初始化.html | 用户点击"上传业务材料"卡片 | `bot_id` |

### 出口定义

| 目标页面 | 触发条件 | 携带数据 |
|---------|---------|---------|
| AI业务分析中.html | 所有文件上传成功后自动跳转 | `bot_id`, `task_id` |

### 页面状态及流转条件

```
[空态] --(点击上传/拖拽)--> [上传中]
[空态] --(拖拽悬停)--> [拖拽悬停]
[拖拽悬停] --(放开文件)--> [上传中]
[上传中] --(上传成功)--> [上传完成]
[上传中] --(上传失败)--> [上传失败]
[上传完成] --(点击开始分析)--> [跳转至AI业务分析中]
[上传完成] --(已达3个文件)--> [满额]
```

| 状态 | 触发条件 | UI 表现 |
|-----|---------|---------|
| 空态 | 未上传文件 | 展示上传引导区域 |
| 拖拽悬停 | 文件拖入区域 | 蓝色边框 + 渐变背景 |
| 上传中 | 文件上传中 | 进度条动画 |
| 上传完成 | 文件上传成功 | 绿色勾选 + 文件卡片 |
| 上传失败 | 验证失败或网络错误 | 红色错误提示 |
| 满额 | 已上传 3 个文件 | 上传区域禁用，显示"已达上限" |

## 数据流

### API 一：上传业务材料

```
POST /api/v1/materials/upload
```

**请求头**：`Authorization: Bearer {tenant_token}`
**Content-Type**：`multipart/form-data`

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | File | 是 | 上传文件（PDF/WORD/MD/TXT） |
| bot_id | string | 是 | BOT ID |

**响应结构**：

```json
{
  "code": 200,
  "data": {
    "material_id": "MAT-202401-001",
    "file_name": "产品手册.pdf",
    "file_size": 2097152,
    "file_type": "application/pdf",
    "storage_url": "https://cdn.example.com/materials/MAT-202401-001.pdf",
    "upload_status": "completed"
  }
}
```

**字段映射**：

| 响应字段 | 前端展示字段 | 类型 | 说明 |
|---------|-------------|------|------|
| material_id | 材料 ID | string | 系统生成唯一 ID |
| file_name | 文件名称 | string | 用户选择文件时读取 |
| file_size | 文件大小 | number | 前端字节数 |
| file_type | 文件类型 | string | MIME 类型 |
| storage_url | 存储 URL | string | OSS/S3 返回地址 |
| upload_status | 上传状态 | string | pending / uploading / completed / failed |

**响应码说明**：

| 响应码 | 说明 | 处理方式 |
|-------|------|---------|
| 200 | 上传成功 | 显示文件卡片，绿色勾选 |
| 400 | 文件格式不支持或超出大小限制 | Toast「不支持的文件格式，请上传 PDF/WORD/MD/TXT」或「文件大小超出限制，最大支持 5MB」 |
| 401 | 未登录 | 跳转登录页面，Toast「请先登录」 |
| 500 | 服务异常 | Toast「服务暂时不可用，请稍后再试」 |

### API 二：提交分析任务

```
POST /api/v1/materials/submit
```

**请求头**：`Authorization: Bearer {tenant_token}`

**请求体**：

```json
{
  "bot_id": "BOT-001",
  "material_ids": ["MAT-202401-001", "MAT-202401-002"]
}
```

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| bot_id | string | 是 | BOT ID |
| material_ids | array | 是 | 已上传材料 ID 列表 |

**响应结构**：

```json
{
  "code": 200,
  "data": {
    "task_id": "TASK-202401-001",
    "status": "processing",
    "redirect_to": "/ai-analysis.html?bot_id=BOT-001&task_id=TASK-202401-001"
  }
}
```

**字段映射**：

| 响应字段 | 前端展示字段 | 类型 | 说明 |
|---------|-------------|------|------|
| task_id | 任务 ID | string | AI 分析任务 ID |
| status | 状态 | string | processing |
| redirect_to | 跳转地址 | string | 跳转至 AI业务分析中页面 |

**响应码说明**：

| 响应码 | 说明 | 处理方式 |
|-------|------|---------|
| 200 | 成功 | 跳转至 AI业务分析中页面 |
| 400 | 参数错误 | Toast「请至少上传 1 个文件」 |
| 401 | 未登录 | 跳转登录页面 |
| 500 | 提交失败 | Toast「分析任务提交失败，请稍后重试」 |

## 处理逻辑

### 场景一：文件上传（点击方式）

**触发条件**：用户点击上传区域触发文件选择器。

**执行逻辑**：
1. 点击上传区域，打开系统文件选择器
2. 选中文件后验证格式（PDF/WORD/MD/TXT）和大小（单文件最大 5MB）
3. 验证通过后添加到上传队列
4. 上传完成后显示文件列表（文件名、大小、状态）

**边界条件**：
- 文件格式不支持时提示错误
- 文件超过 5MB 时提示错误
- 已达 3 个文件上限时禁用上传

**状态变更**：`空态` → `上传中` → `上传完成`

**输出**：文件进入上传队列。

### 场景二：文件上传（拖拽方式）

**触发条件**：用户将文件拖拽至上传区域。

**执行逻辑**：
1. 监听 dragover 事件，显示蓝色边框 + 渐变背景 dragover 状态
2. 放开文件后自动触发文件选择
3. 后续流程同点击方式

**边界条件**：
- 拖拽非文件对象时不响应

**状态变更**：`空态` → `拖拽悬停` → `上传中` → `上传完成`

**输出**：拖拽文件进入上传队列。

### 场景三：多文件上传

**触发条件**：用户选择多个文件。

**执行逻辑**：
1. 支持一次选择多个文件，逐个添加到文件列表
2. 每个文件独立验证格式和大小
3. 最多上传 3 个文件
4. 上传进度独立显示

**边界条件**：
- 超过 3 个文件时只取前 3 个
- 任一文件验证失败不影响其他文件上传

**状态变更**：`空态` → `上传中` → `上传完成` / `满额`

**输出**：多个文件并行上传。

### 场景四：文件上传提交

**触发条件**：用户点击"开始分析"按钮。

**执行逻辑**：
1. 校验至少上传了 1 个文件
2. 校验所有文件均已上传完成
3. 调用文件上传接口，传递 file_ids
4. 后端存储文件元数据至数据库，返回存储 URL
5. 触发 AI 文本提取任务
6. 跳转到 AI 业务分析中页面

**边界条件**：
- 未上传文件时禁用"开始分析"按钮
- 有文件上传失败时提示用户处理

**状态变更**：`上传完成` → 跳转至 AI业务分析中

**输出**：文件上传完成，跳转至分析处理页。

## 异常兜底

| 异常场景 | 检测方式 | 后端处理 | 前端表现 |
|---------|---------|---------|---------|
| 文件格式不支持 | 文件类型校验 | 返回 400 | 错误提示，Toast「不支持的文件格式，请上传 PDF/WORD/MD/TXT」 |
| 文件超过 5MB | 文件大小校验 | 返回 400 | 错误提示，Toast「文件大小超出限制，最大支持 5MB」 |
| 上传数量已达上限 | 文件数量校验 | 返回 400 | 上传区域禁用，Toast「已上传 3 个文件，达上传上限」 |
| 网络超时 | 请求超时 | 返回 504 | Toast「网络连接异常，请检查网络后重试」 |
| 服务不可用 | code 500 | 返回 500 | 全局错误处理，Toast「服务暂时不可用，请稍后再试」 |
| 未登录 | code 401 | 返回 401 | 跳转登录页面，Toast「请先登录」 |
| 上传失败 | 接口返回非 200 | 返回对应错误码 | 红色错误提示，可重试 |
| 分析任务提交失败 | submit 接口返回非 200 | 返回对应错误码 | Toast「分析任务提交失败，请稍后重试」 |
