## 概述

业务初始化是 LeadFinder 业务配置流程的入口页面。用户在完成数据来源选择后，将在此页面选择数据输入方式：上传业务材料（PDF/WORD/MD/TXT）或授权独立站（适用于已有建站系统的企业）。选择完成后跳转至对应的数据输入页面。

|------|---------|---------|

## 功能逻辑

### 场景一：检查初始化状态

**触发条件**：用户进入业务初始化页面。

**执行逻辑**：
1. 前端调用初始化状态接口获取租户信息
2. 检查租户是否已选择数据来源
3. 若已选择，直接跳转对应页面
4. 若已选择但无独立站，授权卡片置灰禁用

**输出**：根据状态展示可选或已选路径。

---

### 场景二：选择数据来源

**触发条件**：用户选择数据输入方式。

**执行逻辑**：
1. 页面展示两张选择卡片：
   - 上传业务材料：点击进入上传业务材料.html，适用于无独立站或材料更全面的企业
   - 授权独立站：点击进入独立站授权.html，适用于已使用建站系统的企业（需已有独立站）
2. 用户点击卡片后，前端发送数据来源选择请求
3. 后端记录用户选择，设置租户数据来源标记（后续不可更改）
4. 返回跳转路径，前端跳转至对应页面

**输出**：跳转至选定的数据输入页面。

## 数据流

### 请求一：获取初始化状态

```
GET /api/v1/tenant/init-status
```

**请求头**：`Authorization: Bearer {tenant_token}`

**响应体**：

```json
{
  "code": 200,
  "data": {
    "tenant_id": "tenant_001",
    "company_name": "华贸盛世科技",
    "has_site": true,
    "selected_source": "upload",
    "bot_init_status": "initialized"
  }
}
```

### 请求二：选择数据来源

```
POST /api/v1/tenant/data-source/select
```

**请求头**：`Authorization: Bearer {tenant_token}`

**请求体**：

```json
{
  "source": "upload"
}
```

**响应体**：

```json
{
  "code": 200,
  "data": {
    "source": "upload",
    "redirect_to": "/upload-materials.html?bot_id=BOT-001"
  }
}
```

| source 值 | 含义 |
|----------|------|
| upload | 上传业务材料 |
| site | 授权独立站 |

## 边界规则

| 规则 | 条件 | 系统行为 |
|------|------|---------|
| 数据来源锁定 | 选择完成后 | 数据来源标记不可更改，不可重复选择 |
| 无独立站 | has_site = false | 授权独立站卡片置灰禁用 |
| 已选择来源 | 已选择但需返回 | 直接跳转至对应页面，不再显示选择页 |
| 租户未登录 | 无效 token | 跳转至登录页面 |
| 超时返回 | 30 分钟无操作 | 提示会话超时，需重新登录 |

---

## 异常处理

| 异常场景 | 检测方式 | 前端表现 | Toast 文案 |
|---------|---------|---------|-----------|
| 租户未登录 | code 401 | 跳转登录页 | 「请先登录」 |
| 租户无独立站 | has_site = false | 授权卡片置灰 | 「您暂无独立站，请选择上传材料」 |
| 已选择数据来源 | selected_source 非空 | 自动跳转 | - |
| 数据来源选择失败 | code 非 200 | 错误提示 | 「数据来源选择失败，请重试」 |
| 服务异常 | code 500 | 全局错误处理 | 「服务暂时不可用，请稍后再试」 |
| 网络超时 | 请求超时 | 错误提示 | 「网络连接异常，请检查网络后重试」 |

## 字段映射

| 前端展示字段 | 后端字段名 | 数据来源说明 |
|-------------|-----------|------------|
| 租户 ID | tenant_id | 系统生成 |
| 公司名称 | company_name | 租户注册信息 |
| 是否有独立站 | has_site | 建站系统查询结果 |
| 已选数据来源 | selected_source | 已选择则直接跳转 |
| BOT 初始化状态 | bot_init_status | initialized / pending / failed |
| 跳转路径 | redirect_to | 后端根据选择返回 |
