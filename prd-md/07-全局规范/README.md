## 模块说明

本目录存放所有模块共用的全局规范文档，涵盖响应码体系、Credits 计费规则、AI 异常处理和数据保留策略。所有功能文档和实现指南均应引用本目录中的规范，确保全局一致性。

## 文档索引

| 文档 | 说明 | 适用节点 |
|------|------|---------|
| `01-全局响应码.md` | HTTP/业务响应码、Toast 规范、401 跳转处理 | P01-P80 |
| `02-Credits余额规则.md` | 冻结结算机制、各场景消耗标准、余额限制 | P01-P80 |
| `03-AI异常处理策略.md` | 5 类 AI 异常、自动重试、降级链路、模型选型 | P08-P46 |
| `04-数据保留策略.md` | 各类数据保留时长、归档清理机制、账号停用处理 | P01-P80 |

## 使用说明

在功能 PRD 文档中引用全局规范时，使用相对路径，例如：

```markdown
 Credits 消耗规则详见 [Credits 余额规则](../07-全局规范/02-Credits余额规则.md)
```
