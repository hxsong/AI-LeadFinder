## 产品定义

**AI-LeadFinder** 是面向 B2B 外贸企业的**独立站全链路 AI Agent 获客系统**。它集成了"AI 数据搜客"、"AI 访客分析"与"AI 智能触达"三大核心模块，实现从流量获取到线索转化的全自动化闭环。

---

## 核心业务流程

**静态数据获客阶段：** AI智能业务分析潜在客户画像 → 全网搜寻潜在企业客户 → AI静态数据分析并打分 → 人工选择客户，挖掘企业联系人 → AI生成静态线索分析报告 → 进入静态线索池 → 销售负责人分配线索进入智能触达阶段。

**动态数据获客阶段：** 访客访问独立站 → 系统以IP为维度采集动态行为数据 → AI动态数据分析并打分 → 人工选择访客IP挖掘企业联系人（UTM标识自动关联或IP解析服务识别） → AI生成动态线索分析报告 → 进入动态线索池 → 进入智能触达阶段。

**全景画像分析阶段：** AI自动关联同企业域名的静动态数据 → 生成全景画像分析报告 → 进入全景线索池 → AI生成触达计划 → 进入智能触达阶段。

**智能触达阶段：** AI分析线索报告和历史消息 → 生成触达计划建议 → 人工选择配置项 → AI生成个性化内容（开发信、社媒消息） → 人工确认 → 通过三方通道发送 → 反馈结果通知销售人员。

**优化迭代阶段：** 统计全流程数据 → 人工提交AI模型分析 → 制定优化策略 → 调整系统参数配置。

**完全自动化阶段：** 系统优化效果合格后，客户可开启完全自动化模式 → 系统自动运行全流程 → 随时可关闭进入手动模式。

## 参与角色

| 角色名称 | 缩写 | 职责描述 |
|----------|------|----------|
| 平台管理员 | SAD | 负责平台租户管理、权限配置、系统监控及模型和提示词维护 |
| 企业管理员 | EBA | 负责配置企业业务信息、操作系统、获取线索 |
| 独立站访客 | VST | 访问企业独立站，产生交互行为 |

---

## 应用系统

| 系统名称 | 缩写 | 核心功能 |
|----------|------|----------|
| 平台运营系统（服务商系统） | SYS | 上架服务和定价，开通管理租户账号、配置权限和资源包 |
| 数据获客主系统 | CLS | 搜索获客、访客雷达、企业配置、线索管理、消息推送、AI内容生成 |
| 独立站系统 | WBS | 获取访客流量、植入埋点代码、集成获客插件 |

---

## 自有服务

| 服务名称 | 缩写 | 核心功能 |
|----------|------|----------|
| 企业分析服务 | EAI | 对企业材料/独立站授权内容进行业务分析，识别行业、上下游、潜在客户画像 |
| 静态数据分析服务 | SDA | 基于客户企业画像全网搜客，清洗和结构化处理，输出匹配度评分 |
| 动态数据分析服务 | DDA | 基于访客行为数据进行动态评分和意向分析 |
| 全景画像分析服务 | PPA | 关联静动态数据，生成企业全景画像报告和综合评分 |
| 智能触达服务 | ITA | 生成触达计划和个性化营销内容 |
| 自动化执行引擎 | AEE | 根据配置的工作流自动执行获客、分析、触达等任务 |

## 三方服务

| 服务名称 | 缩写 | 对接用途 |
|----------|------|----------|
| 三方营销服务 | TMP | EMS/社媒消息触达 |
| 三方数据服务 | TDS | 获取海关数据、社媒数据、地图数据 |
| IP解析服务 | IRS | 通过IP反查识别企业信息 |

## 非功能性需求（摘要）

详见完整需求文档。以下为关键基线：

| 类别 | 指标 |
|------|------|
| 常规页面首屏加载 | ≤ 2 秒 |
| 列表分页/筛选/搜索 | ≤ 1 秒 |
| 按钮点击响应 | ≤ 500 ms |
| AI 业务分析最大时长 | 5 分钟（超时自动跳过） |
| 同时AI业务分析任务 | 1个/单BOT |
| 同时搜客任务 | 1个/单BOT |
| 同时静/动态线索挖掘任务 | 5个/单BOT |
| 平台整体可用性（月度） | ≥ 99.5% |
| Credits账务一致性 | 100% 强一致 |
| 线索数据持久化成功率 | ≥ 99.99% |
