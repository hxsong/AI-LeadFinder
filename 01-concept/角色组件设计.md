# 角色组件设计

**版本：** v1.0.0
**最后更新：** 2026-04-17

---

## 角色定义

| 角色名称 | 缩写 | 职责边界 |
|----------|------|----------|
| 平台管理员 | SAD | 平台租户管理、权限配置、系统监控、AI 模型和提示词维护 |
| 企业管理员 | EBA | 配置企业业务信息、操作系统、全流程获客和线索管理 |
| 独立站访客 | VST | 访问企业独立站，产生交互行为（匿名，无系统账号） |

> 注："独立站访客（VST）"不是系统用户，不登录本系统，其行为数据通过 JS 探针被动采集后进入 EBA 的分析流程。

---

## 角色权限边界

### 平台管理员（SAD）权限

- 查看全局漏斗分析数据
- 下载全量业务数据表进行离线分析
- 调优 AI 模型参数与提示词策略并下发至各租户
- 审核租户订单、开通账号、分配资源包
- 配置全局共享资源池（全局 Credits）

### 企业管理员（EBA）权限

- 为当前 BOT 上传业务材料或授权独立站
- 提交 AI 业务分析请求（单 BOT 同时 1 个任务）
- 输入搜客需求，执行搜客任务（单 BOT 同时 1 个任务）
- 挖掘静态/动态线索（单 BOT 同时最多 5 个任务）
- 确认触达方案并执行发送
- 查看业务漏斗数据（当前 BOT 范围）
- 开启/关闭完全自动化执行模式

**EBA 不可操作：** 平台级配置、AI 模型调优、其他租户数据

## 系统级组件（EBA 可操作界面）

以下为 EBA 在数据获客主系统（CLS）中的主要操作界面：

### 账号与资源模块

| 页面 | 说明 |
|------|------|
| 租户资源充值 | 账户余额管理、Credits 充值 |
| 资源下发确认 | 资源包到账确认 |
| 多BOT工作台 | 多 BOT 实例切换与管理 |
| 业务初始化 | 上传材料/独立站授权、AI 业务分析触发 |
| 上传业务材料 | 文件上传（PDF/WORD/MD/TXT，单次最多3个，单文件≤5MB） |
| 独立站授权 | 选择核心主站授权（1个/单BOT） |
| AI业务分析中 | 分析任务执行状态 |
| 业务分析报告 | ICP 建模结果展示 |

### 搜索客户模块

| 页面 | 说明 |
|------|------|
| 搜客需求输入 | 输入自然语言描述指定搜客条件（目标公司数默认100家，上限200家） |
| 搜客策略预览 | AI 生成的搜客策略和预估 Credits 消耗 |
| 搜客执行中 | 搜客任务执行状态 |
| 搜客记录 | 搜客历史记录 |
| 搜客结果列表 | 企业数据分析结果列表，含评分展示 |

### 访客雷达模块

| 页面 | 说明 |
|------|------|
| 探针部署 | 获取 JS 探针代码，支持一键安装 |
| 访客雷达 | 所有访问记录列表，支持 IP 行为分析 |
| 访客行为详情 | 特定 IP 的行为分析和历史记录 |

### 线索模块

| 页面 | 说明 |
|------|------|
| 静态线索池 | 静态线索分析报告列表，支持销售分配 |
| 静态线索详情 | 静态线索报告详情，含联系人信息和触达计划建议 |
| 动态线索池 | 动态线索分析报告列表，支持销售分配 |
| 动态线索详情 | 动态线索报告详情，含意向评分和触达建议 |

---

## 组件复用原则

- 列表页组件（线索池、搜客结果、访客雷达）统一使用相同的筛选/分页/导出组件
- 详情页统一使用"报告卡片 + 数据表格 + 操作按钮"三段式布局
- 评分展示统一使用"综合评分 + 四维雷达图 + 数据完整度"格式
- 表单页面统一使用步骤条（Stepper）引导用户完成配置流程

---

## 自动化与手动模式

| 模式 | 说明 |
|------|------|
| 手动模式 | 人工逐项确认后执行（默认） |
| 完全自动化模式 | EBA 开启后，系统自动循环执行搜客、分析、触达闭环 |

EBA 可随时关闭自动化模式回到手动模式。
