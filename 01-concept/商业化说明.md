## 商业模式

采用 **"SaaS 订阅 + 用量付费"** 的混合模式。

- **订阅费（门槛）：** 降低入门门槛（如 3980 元/年），包含基础的访客识别功能，快速圈占市场用户。
- **用量费（增收）：** 针对高端用户，对"AI 线索深度挖掘次数"、"AI 邮件生成量"等高价值功能设置消耗点数（Credits），对标 ZoomInfo 的 Credit 模式但价格更亲民。

## 潜在客户评分框架

### 核心问题

在 B2B 获客场景中，如何判断一个企业是否是潜在客户？

```
❌ 错误思路：找和我相似的企业
   轮胎厂找轮胎厂 → 同行，不是客户

✅ 正确思路：找需要我产品的企业
   轮胎厂找汽修厂 → 下游客户
```

### 四维评分框架

| 维度 | 说明 | 数据来源 |
|------|------|----------|
| 需求匹配度 | 企业是否处于客户下游 | AI 模型供需关系判断 |
| 购买能力 | 企业是否有购买能力 | 地图数据 + 网络抓取 |
| 采购意向 | 企业是否有采购行为 | 海关数据 |
| 活跃意向 | 企业是否市场活跃 | 展会数据 |

### 动态权重生成

权重根据客户业务类型动态生成，而非固定。

| 业务类型 | 需求匹配度 | 购买能力 | 采购意向 | 活跃意向 |
|----------|-----------|----------|----------|----------|
| 外贸型 | 0.32 | 0.20 | 0.35 | 0.13 |
| 生产型 | 0.40 | 0.25 | 0.20 | 0.15 |
| 服务型 | 0.35 | 0.25 | 0.15 | 0.25 |
| 渠道型 | 0.35 | 0.20 | 0.30 | 0.15 |

### 综合评分计算

```
潜在客户得分 = 需求匹配度 × W1
             + 购买能力 × W2
             + 采购意向 × W3
             + 活跃意向 × W4

其中 W1 + W2 + W3 + W4 = 1.0
```

## 评分细则摘要

### 需求匹配度（AI 模型计算）

通过 AI 模型判断候选企业是否是客户的潜在客户。AI 基于客户业务信息和候选企业信息，推理供需关系并给出匹配度分数。

**Prompt 判断逻辑：**
1. 供需关系分析：候选企业是否需要客户的产品/服务？
2. 品类匹配分析：候选企业经营的品类是否与客户的产品相关？
3. 业务模式匹配：候选企业的业务类型是否与客户的目标客户类型匹配？
4. 市场范围匹配：候选企业的市场范围是否与客户的市场范围有交集？

**推荐模型：** `gpt-4o-mini` 批量计算，`gpt-4o` 对高分企业二次验证。

### 购买能力（规模 + 地理）

| 规模指标 | 得分 | 地理指标 | 得分 |
|----------|------|----------|------|
| 大型企业（500人以上） | 60 | 在目标区域内 | 40 |
| 中型企业（50-500人） | 45 | 不在目标区域内 | 0 |
| 小型企业（10-50人） | 30 | 无数据 | 20（默认） |
| 微型企业（10人以下） | 15 | | |
| 无数据 | 30（默认） | | |

```
购买能力得分 = (规模得分 × 0.6) + (地理得分 × 0.4)
```

### 采购意向（海关记录）

| 海关记录情况 | 得分 |
|-------------|------|
| 有对口 HS Code 进口记录（近12个月） | 100% |
| 有相关 HS Code 进口记录（近12个月） | 70% |
| 有进口记录但 HS Code 不匹配 | 40% |
| 无海关记录 | 0% |

时间衰减：0-30天=1.0，31-90天=0.95，91-180天=0.85，181-365天=0.70，超过365天=0.50。

### 活跃意向（展会记录）

| 展会记录情况 | 得分 |
|-------------|------|
| 参展商身份参加行业展会（近12个月） | 100% |
| 观众身份参加行业展会（近12个月） | 60% |
| 参加非行业展会 | 30% |
| 无展会记录 | 0% |

## 数据完整度

系统展示评分时同时展示数据完整度：

```
评分：77.64 分
数据完整度：75%（海关数据 ✓ | 展会数据 ✓ | 规模数据 ✓ | 官网数据 ✗）
```

| 数据缺失情况 | 处理策略 |
|-------------|----------|
| 无海关数据 | 采购意向得分 = 0，但不排除该企业 |
| 无展会数据 | 活跃意向得分 = 0，但不排除该企业 |
| 无规模数据 | 购买能力得分 = 默认中等（30分） |
| 无官网数据 | 需求匹配度得分 = 仅基于企业类型判断 |
