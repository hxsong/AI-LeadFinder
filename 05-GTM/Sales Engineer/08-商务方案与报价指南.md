# AI LeadFinder 商务方案与报价指南

## 商业模式

采用 **"系统服务费 + 资源购买"** 的灵活模式，所有功能无版本限制，客户按需购买 BOT 席位和 Credits 资源。

***

## 费用结构

### 1. 系统服务费（按年/BOT计费）

系统服务费是维持账号使用的基础费用，按年收取，与 BOT 席位数量挂钩。

| 计费项       | 价格            | 说明                   |
| --------- | ------------- | -------------------- |
| **系统服务费** | 3,000 元/年/BOT | 包含系统使用权限、全部功能开放、技术支持 |

**重要说明：**

- 系统服务费按年计费，到期后系统入口关闭，所有功能不可用
- BOT 席位本身无到期时间，但受系统服务期限制
- 系统到期后，已购买的 Credits 不会清除，但无法使用（需续费系统服务后才能恢复）
- 新增 BOT 席位 = 增加系统服务费基数（3,000 元/年/BOT）

### 2. Credits 资源包

Credits 是消耗型资源，用于支付 API 调用和 AI 服务成本，**永久有效，不清零**。

| 资源包类型 | Credits 额度 | 单价           |
| ----- | ---------- | ------------ |
| 基础包   | 1,000      | 1,000 元      |
| 标准包   | 5,000      | 4,500 元（9折）  |
| 专业包   | 20,000     | 16,000 元（8折） |
| 企业包   | 100,000    | 70,000 元（7折） |

## Credits 消耗标准

| 功能场景      | 计费规则           | 说明                      |
| --------- | -------------- | ----------------------- |
| **全网搜客**  | 按实际 API 调用次数计费 | 预估调用 → 后台折算 → 实际执行后多退少补 |
| **IP 解析** | 3 Credits/次    | 解析访客 IP 对应的企业信息         |
| **联系人挖掘** | 2 Credits/有效邮箱 | 获取 1 个带验证邮箱的联系人         |
| **附加社媒**  | +1 Credit/个    | LinkedIn、Twitter/FB 等   |

### 计费原则

- **透明预估算费**：执行前展示预估 Credits 消耗
- **实际结算**：按真实 API 成功调用次数核算
- **多退少补**：预估时锁定的多余 Credits 退回账户
- **保底策略**：未查到邮箱/数据时，扣费为 0

## 报价策略

### 定价原则

1. **按需付费**：客户根据实际业务需求购买 BOT 和 Credits
2. **灵活组合**：系统服务费 + BOT 席位 + Credits 可独立调整
3. **量大从优**：套餐购买享受阶梯折扣
