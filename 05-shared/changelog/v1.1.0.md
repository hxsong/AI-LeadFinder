### 双向索引映射表建立

#### PRD 双向索引体系

在每份功能说明文档头部建立标准化的 PRD 头部元数据块，实现双向可追溯：

**正向索引**（功能说明 → PRD 节点）：
```markdown
> **对应 PRD**：[PRD节点编号] — [PRD节点名称]
> **功能说明文档**：[prototype/docs/功能名-说明.html]
```

**反向索引**（PRD 节点 → 功能说明）：
在 PRD 文档中引用对应的功能说明文档路径，实现交叉定位。

**全局索引映射表**：
创建 `docs/双向索引映射表.md`，集中展示：
- 所有功能说明文档与原型页面的对应关系
- 所有功能说明文档与 PRD 节点的对应关系
- 所有功能说明文档与修订日志的对应关系
