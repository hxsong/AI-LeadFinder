Usage Scenarios

按场景快速定位

不同销售阶段应使用的资料组合

销售拜访前准备

01-产品一页纸简介 + 04-目标客户画像

确认客户匹配度（高成功概率画像），准备针对性的 Elevator Pitch

销售拜访前准备

1. 阅读 01-产品一页纸简介 和 04-目标客户画像
2. 确认客户匹配度（高成功概率画像）
3. 准备针对性的 Elevator Pitch

初次客户沟通

01-产品一页纸简介 → 02-产品价值主张 → 03-产品功能架构

用一页纸简介开场，根据客户痛点引用对应价值，如需深入了解展示功能架构

初次客户沟通

1. 用 01-产品一页纸简介 开场
2. 根据客户痛点引用 02-产品价值主张 中的对应价值
3. 如需深入了解，展示 03-产品功能架构

方案演示/POC

07-演示指南 + 06-典型场景方案

参考演示指南执行标准化演示，结合典型场景讲客户最相关的场景，使用ROI计算器帮客户算经济账

方案演示/POC

1. 参考 07-演示指南与话术脚本 执行标准化演示
2. 结合 06-典型场景与ROI方案 讲客户最相关的场景
3. 使用 ROI 计算器帮客户算经济账

客户比价阶段

05-竞品对比分析 + 02-产品价值主张 + 06-典型场景方案

查阅竞品对比分析中的应对策略，强调"3个唯一"差异化，准备对比数据

客户比价阶段

1. 查阅 05-竞品对比分析 中的对应竞品应对策略
2. 强调 02-产品价值主张 中的"3 个唯一"差异化
3. 准备 06-典型场景与ROI方案 中的对比数据

商务谈判阶段

08-商务方案与报价指南 + 09-实施路线图 + 10-安全与合规FAQ

参考报价指南制定报价方案，说明上线周期和价值实现时间，准备安全FAQ应对企业级客户顾虑

商务谈判阶段

1. 参考 08-商务方案与报价指南 制定报价方案
2. 使用 09-实施路线图 说明上线周期和价值实现时间
3. 准备 10-安全与合规FAQ 应对企业级客户的安全顾虑

签约后交付

09-实施路线图 + 04-目标客户画像

与客户共同确认实施路线图，设置成功指标基准，定期回顾目标达成情况

签约后交付

1. 与客户共同确认 09-实施路线图
2. 设置 04-目标客户画像 中的成功指标基准
3. 定期回顾目标达成情况
