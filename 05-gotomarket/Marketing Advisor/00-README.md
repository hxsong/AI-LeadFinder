# AI LeadFinder - 市场推广部 GTM 材料

## 材料目录

本目录包含面向市场推广部门的GTM材料，用于支持品牌传播、内容营销、活动运营等市场工作。

| 序号 | 文档名称 | 用途 | 目标读者 |
|------|----------|------|----------|
| 01 | 品牌定位与调性.md | 品牌核心信息、语言风格 | 市场全员 |
| 02 | 产品卖点提炼.md | 核心卖点、传播话术 | 内容/运营 |
| 03 | 内容营销指南.md | 内容策略、选题方向 | 内容团队 |
| 04 | 社交媒体运营.md | 社媒策略、发布规范 | 社媒运营 |
| 05 | 活动运营手册.md | 线上线下活动 SOP | 活动运营 |
| 06 | 广告投放指南.md | 广告策略、素材规范 | 投放团队 |
| 07 | SEO/SEM 策略.md | 搜索营销策略 | 增长团队 |
| 08 | 公关传播手册.md | PR 策略、媒体关系 | PR 团队 |

## 使用指南

1. **品牌传播**：参考 01-02，确保信息一致性
2. **内容创作**：参考 03
3. **社媒运营**：参考 04，使用统一规范
4. **活动策划**：参考 05，使用标准 SOP
5. **广告投放**：参考 06-07，优化投放效果
6. **公关传播**：参考 08，维护品牌形象
