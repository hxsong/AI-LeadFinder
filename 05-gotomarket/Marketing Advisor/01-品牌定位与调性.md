## 品牌调性

### 品牌人格

**AI LeadFinder 的品牌人格：专业、智能、高效、可信赖**

```
                    专业
                      ▲
                     /│\
                    / │ \
                   /  │  \
        可信赖 ◄──────┼──────► 高效
                   \  │  /
                    \ │ /
                     \│/
                      ▼
                    智能
```

### 调性维度详解

| 维度 | 是 | 不是 |
|------|-----|------|
| **专业** | 数据驱动、方法论、行业洞察 | 空洞口号、过度承诺 |
| **智能** | AI 技术、自动化、智能决策 | 人工替代、冷冰冰的机器 |
| **高效** | 快速见效、节省时间、提升产出 | 急功近利、牺牲质量 |
| **可信赖** | 透明、安全、客户成功导向 | 隐瞒风险、过度推销 |

### 语言风格指南

#### 语气

- **正式但友好**：专业但不生硬
- **积极正向**：强调价值而非问题
- **数据支撑**：用数字说话

#### 用词偏好

| 推荐用词 | 避免用词 |
|----------|----------|
| 赋能 | 替代 |
| 提升 | 颠覆 |
| 智能 | 人工智能（过于技术） |
| 获客 | 拉新（过于互联网） |
| 线索 |  leads（中英混杂） |
| 效率 | 效能（过于抽象） |

#### 句式风格

**推荐**：
- "AI LeadFinder 帮助您..."
- "通过 AI 技术，实现..."
- "让客户获客效率提升 X 倍"

**避免**：
- "我们是最棒的..."
- "颠覆传统..."
- "秒杀竞品..."

## 品牌禁忌

### 禁止行为

- ❌ 夸大效果（如"保证成交"）
- ❌ 贬低竞品（如"XX 产品很差"）
- ❌ 过度技术化（如堆砌技术术语）
- ❌ 不专业表达（如网络 slang）
- ❌ 误导性信息（如虚假数据）

### 敏感话题

- 避免承诺具体成交数字
- 避免涉及政治敏感内容
- 避免侵犯客户隐私的案例
- 避免未经验证的数据
