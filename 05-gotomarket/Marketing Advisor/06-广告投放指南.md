## 搜索广告策略

**品牌词（20% 预算）**：

**行业词（50% 预算）**：
- B2B 获客工具
- 外贸获客软件
- 海关数据查询
- 独立站访客识别
- 销售线索管理

**长尾词（30% 预算）**：
- B2B 外贸怎么找客户
- 独立站怎么提高转化率
- 海关数据怎么用
- 展会后怎么跟进客户

### 广告创意

**标题公式**：
- 核心词 + 效果承诺
- 核心词 + 免费试用
- 核心词 + 客户数量

**示例**：
```
标题1：B2B 获客工具 | 助力企业提升获客效率 | 免费试用
标题2：外贸获客软件 | 已服务众多企业 | 14 天免费
标题3：海关数据查询 | 覆盖多国数据 | 立即体验
```

**描述**：
```
AI LeadFinder 是 B2B 外贸企业的 AI 获客专家，
通过 AI 技术实现自动化获客，让获客效率提升 5-10 倍。
立即免费试用，无需信用卡。
```

### 落地页策略

**落地页结构**：
1. Hero 区：价值主张 + CTA
2. 痛点共鸣：3 个核心痛点
3. 解决方案：产品功能介绍
4. 效果展示：数据 + 案例
5. 信任背书：客户 Logo + 证言
6. FAQ：常见问题
7. CTA：试用表单

## 信息流广告策略

### 微信朋友圈广告

**广告形式**：图文 / 视频
**定向策略**：
- 基础：25-45 岁，一二线城市
- 兴趣：B2B、外贸、销售、营销
- 行为：关注商业、科技公众号
- 自定义：已有客户 Lookalike

**创意方向**：
- 痛点型："销售大量时间花在找客户？"
- 效果型："提升获客效率的秘密"
- 案例型："某企业成功成交多家客户"

### LinkedIn 广告

**广告形式**：Sponsored Content / Message Ads
**定向策略**：
- 职位：Sales Manager、Marketing Manager、CEO
- 公司规模：50-500 人
- 行业：制造业、贸易、科技
- 地区：目标市场国家

**创意方向**：
- 专业型：数据驱动的获客方法论
- 案例型：客户成功故事
- 教育型：B2B 获客优秀实践

## 广告素材规范

### 图片素材

**尺寸要求**：
- 百度搜索：600×300
- 信息流：800×450 / 1280×720
- 微信朋友圈：800×450

**设计规范**：
- 品牌色为主（#2563EB）
- 突出核心卖点
- 包含 CTA 按钮
- 避免过多文字

### 视频素材

**时长**：15-30 秒
**结构**：
- 0-3秒：痛点钩子
- 3-20秒：解决方案
- 20-30秒：效果展示 + CTA

**规范**：
- 分辨率：1080p
- 格式：MP4
- 大小：< 50MB

## 广告效果监控

### 核心指标

| 指标 | 定义 | 目标值 |
|------|------|--------|
| **CTR** | 点击率 | > 2% |
| **CPC** | 单次点击成本 | < ¥10 |
| **CPM** | 千次展示成本 | < ¥50 |
| **CVR** | 转化率 | > 5% |
| **CPA** | 单次获客成本 | < ¥300 |
| **ROAS** | 广告支出回报率 | > 3:1 |

### 优化策略

**CTR 优化**：
- 测试不同标题
- 优化广告图片
- 调整定向策略

**CVR 优化**：
- 优化落地页
- 简化表单
- 增加信任背书

**CPA 优化**：
- 排除低质量流量
- 优化关键词匹配
- 调整出价策略
